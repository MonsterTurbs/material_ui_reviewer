🚀 What is Heroku?

Heroku is a cloud platform (launched in 2007) that makes it easy to deploy and scale web apps. It handles billions of requests daily! 🌐

👍 Why Choose Heroku?

  • 🔒 Strong security and easy scaling from small to huge traffic.
  • ⚙️ Low-config setup—no server management needed.
  • 💰 Free tier available for learning and small projects.

✏️ Step 1: Sign Up

  • 1️⃣ Go to Heroku.com and click “Sign Up”.
  • 2️⃣ Enter your name, email, and choose role (e.g., student).
  • 3️⃣ Select your country and primary language (Node.js for this project).
  • 4️⃣ Confirm your email and set a password. ✅

🔧 Step 2: Create a New App

  • 📥 Log into the Heroku Dashboard.
  • ➕ Click “New” > “Create new app”.
  • 🆔 Name it (lowercase, dashes instead of spaces), choose region, and create. 🌍

⚙️ Step 3: Configure Buildpack

In your app's Settings tab, add the Node.js buildpack so Heroku knows how to build your Next.js app.

💻 Step 4: Install & Login via CLI

  • 📥 Install the Heroku CLI from their website.
  • 💬 Run heroku login in your terminal and follow prompts. 🔑

📁 Step 5: Prepare Your Project

  • 📂 Duplicate your project folder if you also deploy elsewhere.
  • 📂 cd into your project directory.
  • 🔧 Initialize git: git init.
  • 🔗 Add Heroku remote: heroku git:remote -a your-app-name.

📝 Step 6: Update package.json

  • 🔄 In "scripts", change "start" to "next start -p $PORT" so Heroku uses its port.
  • ➕ Add "heroku-postbuild": "npm run build" for automatic production build. 🏗️

🚢 Step 7: Deploy! 🛳️

  • git add .
  • git commit -m "initial commit"
  • git push heroku master
  • 🎉 Your app builds and goes live at https://your-app-name.herokuapp.com!

✅ Step 8: Verify & Test

  • 🌐 Open your live URL.
  • ✉️ Test forms (Contact Us), buttons, and any features to ensure everything works. 🛠️

🎯 Quick Recap

  • 🚀 Sign up and create an app on Heroku.
  • 🔧 Configure Node.js buildpack.
  • 💻 Install Heroku CLI and login.
  • 📁 Initialize git, connect remote.
  • 🔄 Update scripts, deploy with git push.
  • ✅ Check your live site! 🎉
