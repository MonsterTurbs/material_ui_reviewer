🌟 Why Use a Custom Domain?

Sharing yourapp.com looks more professional than yourapp.herokuapp.com! 😊

💳 Step 1: Add Billing Info

  • Heroku requires a credit card on file to use custom domains (no extra charges!).
  • Go to your Heroku Account Settings > Billing > Add Card. ✔️

📝 Step 2: Add Your Domain via CLI

Run this in your project folder:

heroku domains:add www.yourcustomdomain.com -a your-app-name

This tells Heroku to accept requests for your custom URL. 🚀

🌐 Step 3: Configure DNS (CNAME)

  • Name: www
  • Type: CNAME
  • Value: the DNS target from Heroku (e.g., your-app-name.herokudns.com).

Do this in your domain provider’s DNS settings pane. 🛠️

⏳ Step 4: Wait for Propagation

DNS changes can take up to 48 hours, but often update sooner. ⏱️

✅ Quick Recap

  • 1️⃣ Add card to Heroku billing.
  • 2️⃣ Run heroku domains:add with your domain.
  • 3️⃣ Add CNAME record (www → Heroku DNS target).
  • 4️⃣ Wait for DNS to update—then enjoy your own domain! 🎉
