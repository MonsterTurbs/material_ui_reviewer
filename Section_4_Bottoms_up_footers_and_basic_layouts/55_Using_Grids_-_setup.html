🔧 Footer Grid Layout - Study Notes

1. ➕ Import Grid

In Footer.js, add:

import Grid from '@material-ui/core/Grid';

✅ Now you can use the Grid component!

2. 🚧 Wrap Links in a Container

  • At top of footer, before the image, add:
  • <Grid container>…</Grid>

This makes a grid container for your link items.

3. 🎯 Center Links

To move links to the middle horizontally, use:

<Grid container justify="center">

✨ Links now sit in the center of the footer!

4. 📐 Position Absolutely

If your links sit under the slash image, make them float by adding a class:

.mainContainer { position: absolute; top: 0; }

✅ This lets links overlap the image just right.

5. 🎨 Style Link Text

Give your links a friendly look:

.link {
  color: white;
  font-size: 0.75rem;
  font-weight: bold;
}

🖌️ Now links look clean and readable.

6. 🏷️ Nest Columns

  • Each section (Home, Services, etc.) goes in its own column:
  • <Grid item><Grid container direction="column">…</Grid></Grid>

🗂️ Columns hold links stacked top-to-bottom.

7. ↔️ Add Spacing Between Columns

Use the spacing prop on the outer container:

<Grid container spacing={2}> … </Grid>

📏 Spacing of 2 means 2×8px = 16px gaps.

8. 🎉 Final Layout

  • 🔄 Try resizing the window to see links stay centered.
  • 🤖 Your footer now uses Grid for a flexible, neat layout!
