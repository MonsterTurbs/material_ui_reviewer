🔗 Footer Links & Navigation - Study Notes

1. ➕ Add Spacing to Columns

On each inner grid container, add spacing={2} to get 16px gaps (2 × 8px):

<Grid container direction="column" spacing={2}>…</Grid>

✨ Links now have space around them!

2. 📏 Give Columns Margin

  • Add className={classes.gridItem} to each outer <Grid item>.
  • In styles, set .gridItem {{ margin: '3em' }} for responsive whitespace.

3. 🛠️ Make Links Clickable

Import React Router's Link:

import { Link } from 'react-router-dom';

Then on each grid item, add props:

<Grid
  component={Link}
  to="/services"
  ...
>Services</Grid>

🔗 Now clicking the text navigates to that page!

4. 🚫 Remove Underline

Add textDecoration: 'none' to your .link style so links look clean:

.link {{
  color: 'white',
  fontSize: '0.75rem',
  fontWeight: 'bold',
  textDecoration: 'none'
}}

5. 🎉 Test Your Footer

  • 🔄 Save and refresh your app.
  • 💻 Click each footer link to see it change pages.
  • 📱 Resize the window or check on mobile—layout stays nice!
