💠 Material UI Grid System - Study Notes

✨ What is the Grid? ✨

The Grid helps you place things neatly on the screen and makes them move or resize when the screen changes. It uses CSS Flexbox under the hood!

📦 Containers vs. Items

  • 🧱 Container: Holds and spaces out Items.
  • 📄 Item: A child inside a container. Your content goes here.

↔️ Spacing & the 8px Rule

Use spacing on a Container. The number (0–10) multiplies by 8px. For example:

<Grid container spacing={2}> // 2×8px = 16px gaps</Grid>

🔢 12-Column Layout

Each row has 12 columns. Give each Item a number (1–12) to say how many columns it spans:

  • 12 = full width (100%)
  • 6 = half width (50%)
  • 3 = quarter width (25%)

📱 Breakpoints & Responsive

  • xs (extra-small): mobile phones
  • sm (small): tablets
  • md (medium): small laptops
  • lg (large): desktops
  • xl (extra-large): big screens

Set different column spans for each breakpoint:

<Grid item xs={12} sm={6} md={4}> ... </Grid>

🛠️ Fixed vs. Fluid Layouts

If you give only xs, your layout stays the same on all screens. To change on larger screens, add more breakpoints.
