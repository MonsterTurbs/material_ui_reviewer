🦶 Building a Footer – Study Notes

🏗️ 1. Create the Footer File

  • 📂 In /ui, make a new file named Footer.js.
  • ✅ This will hold our footer component.

🛠️ 2. Import React & makeStyles

  • import React from 'react';
  • import { makeStyles } from '@material-ui/core/styles';

🎨 3. Define Footer Styles

  • ✅ Use makeStyles(theme => ({{ footer: {{ backgroundColor: theme.palette.common.blue, width: '100%' }} }}));
  • 📏 width: 100% makes the footer span the screen.

🧩 4. Build the Footer Component

  • 🔧 Inside Footer.js:
  • const classes = useStyles();
  • return <footer className={classes.footer}>Example Footer</footer>;

📦 5. Render Footer in App

  • import Footer from './ui/Footer';
  • 🔽 Below your <Switch>, add <Footer />.

⚙️ 6. Remove Body Margin

  • 📝 In public/index.html, set <body style="margin:0">.
  • 🖼️ Footer now touches both edges of the screen.

📏 7. Push Content to See Footer

  • 📐 In your Home component, add a style like <div style={height: '2000px'}> to create space.
  • 🔍 Scroll down to confirm the footer sits at the bottom.

➗ 8. Add the Black Slash Image

  • import footerAdornment from '../assets/FooterAdornment.svg';
  • 🔳 Replace placeholder text with <img src={footerAdornment} alt="black slash" className={classes.adornment} />.

📐 9. Style the Adornment

  • 🌈 Add adornment style: width: '25em'.
  • 📱 Make it responsive: use theme.breakpoints.down('md') for width: '21em', and 'xs' for '15em'.
  • ⬇️ Use vertical-align: bottom to flush to footer bottom.

🔒 10. Fix Drawer Clipping

  • ☝️ Footer was behind the drawer.
  • position: 'relative', zIndex: 1302 on classes.footer brings it above.

🎉 11. Final Check

  • 🔄 Resize window, open drawer—footer and header clip nicely.
  • 📱 Test on your phone via local IP for a mobile preview.
