📱 Finish Footer with Social Icons

1. 🎨 Import Your SVG Icons

Bring in your social icons at the top of Footer.js:

import Facebook from '../assets/facebook.svg';
import Twitter  from '../assets/twitter.svg';
import Instagram from '../assets/instagram.svg';

2. 🏗️ Add a Grid Container

Under the slash image, wrap icons in a grid:

<Grid container spacing={2} justify="flex-end" className={classes.socialContainer}>
  …icon items…
</Grid>

3. 📎 Make Each Icon an Item

Inside container, add one item per icon:

<Grid item component="a" href="https://facebook.com" target="_blank" rel="noopener noreferrer">
  <img src={Facebook} alt="Facebook logo" className={classes.icon} />
</Grid>

4. 🖌️ Style Icon Size

Give icons a good size (e.g. 4em):

.icon {
  width: 4em;
  height: 4em;
}

5. 🆙 Position Above Slash

Let icons float over the slash with absolute positioning:

.socialContainer {
  position: absolute;
  bottom: 0;
  margin-top: -6em;
}

6. ↔️ Align to the Right

Use justify="flex-end" on the container to push icons right:

<Grid container justify="flex-end" …>…</Grid>

7. ⚖️ Add Spacing Between Icons

Use spacing={2} for nice gaps (2×8px =16px):

<Grid container spacing={2} …>…</Grid>

8. 📏 Make Icons Responsive

Shrink icons on small screens with a breakpoint:

[theme.breakpoints.down('xs')]: {
  icon: { width: '2.5em', height: '2.5em' }
}

9. 📱 Adjust Position for Mobile

Move icons closer to edge on phones:

[theme.breakpoints.down('xs')]: {
  socialContainer: { right: '0.6em' }
}

🎉 You're Done!

Your footer now has clickable, well-placed social icons that resize perfectly on any device. Great job! 🙌
