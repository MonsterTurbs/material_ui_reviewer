📋 Material UI Menu Component - Study Notes

❓ What is the Menu Component?

The Menu component displays a pop-up list of options. It’s one of the most common UI elements in applications.

🧠 How It Works

Material UI uses the <Menu> and <MenuItem> components to render menus. The menu appears anchored to a given element.

The open prop controls visibility: if anchorEl is not null, open is true, otherwise false.

⚙️ Key Props

  • anchorEl - Element the menu is attached to
  • open - Boolean flag to show or hide the menu
  • onClose - Function called when the menu closes
  • keepMounted - Keeps the menu in the DOM even when hidden
  • aria-controls & aria-haspopup - Accessibility attributes

✅ Managing Selected Items

To track which menu item is selected:

  1. Set up a state variable, e.g. selectedIndex.
  2. Use array.map to render <MenuItem> components.
  3. Add selected={selectedIndex === index} to each item.
  4. Update selectedIndex in an onClick handler.

💡 Use Case in Header

In the app header, hover or click the Services tab to display menu options for subpages like Custom Software, Mobile Apps, and Websites.
