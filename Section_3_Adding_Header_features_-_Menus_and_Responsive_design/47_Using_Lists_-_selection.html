Managing Selection in Drawer

üìù Study notes for marking the active menu item in the Drawer using the selected prop.

1. Add selected Prop

For each <ListItem>, add the selected prop:

<ListItem
  selected={value === 0}
  ...
>
  ...
</ListItem>

2. Use the Same value State

Use the value from your tabs state to track the active route:

const [value, setValue] = useState(0);  // 0 = Home, 1 = Services, ...

3. Update onClick Handlers

Ensure you set the correct index when an item is clicked:

onClick={() => {
  setValue(2);             // For Revolution (index 2)
  setOpenDrawer(false);    // Also close the drawer
}}

4. Dim Unselected Items

In your drawItem style, add:

drawItem: {
  ...theme.typography.tab,
  color: '#fff',
  opacity: 0.7
},

5. Highlight Selected Text

Create a drawItemSelected style:

drawItemSelected: {
  opacity: 1
},

6. Conditional Class for Text

Use a conditional array for the className of <ListItemText>:

<ListItemText
  disableTypography
  className={
    value === i
      ? [classes.drawItem, classes.drawItemSelected].join(' ')
      : classes.drawItem
  }
>
  {label}
</ListItemText>

7. Verify Behavior

Test in browser and on mobile: select different items and refresh to ensure the selection persists.
