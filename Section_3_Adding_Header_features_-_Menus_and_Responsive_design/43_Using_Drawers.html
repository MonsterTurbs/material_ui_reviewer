🔄 Responsive Drawer Implementation

🚀 Beginner-friendly steps to replace header tabs with a Swipeable Drawer in Material UI.

1. 🖥️ Install Icon Package

📦 Run npm to install Material UI icons:

npm install @material-ui/icons

2. 📂 Import Components

⬇️ Import SwipeableDrawer and MenuIcon:

import SwipeableDrawer from '@material-ui/core/SwipeableDrawer';
import MenuIcon from '@material-ui/icons/Menu';

3. 🍏 Detect iOS Devices

🔍 Use the official snippet to detect iOS for smoother transitions:

const iOS = /iPad|iPhone|iPod/.test(navigator.userAgent);
const disableSwipeDiscovery = iOS;
const disableTransition = iOS;

4. 🛠️ Setup Drawer State

🔧 Create a state hook to control drawer visibility:

const [openDrawer, setOpenDrawer] = useState(false);

5. 🔄 Refactor State Names

✏️ Rename handlers for clarity:

const [openMenu, setOpenMenu] = useState(false);
// Use setOpenMenu(true) / setOpenMenu(false) in handlers

6. 📐 Create Drawer Component

📐 Wrap the drawer JSX in a fragment and add props:

const drawer = (
  <>
    <SwipeableDrawer
      disableBackdropTransition={disableTransition}
      disableDiscovery={disableSwipeDiscovery}
      open={openDrawer}
      onOpen={() => setOpenDrawer(true)}
      onClose={() => setOpenDrawer(false)}
    >
      {/* Example Drawer Content */}
    </SwipeableDrawer>
  
);

7. ☝️ Add IconButton & MenuIcon

🔁 Import IconButton and toggle the drawer:

import IconButton from '@material-ui/core/IconButton';

<IconButton
  disableRipple
  onClick={() => setOpenDrawer(!openDrawer)}
>
  <MenuIcon className={classes.drawIcon} />
</IconButton>

8. 🔀 Conditional Rendering

⚖️ Swap between drawer and tabs based on matches:

{'{'}matches ? drawer : tabs{'}'}

9. 📏 Style Drawer Icon

🎨 Use JSS to remove hover bg, right-align, and resize:

drawIconContainer: {
  '&:hover': { backgroundColor: 'transparent' },
  marginLeft: 'auto'
},

drawIcon: {
  width: '50px',
  height: '50px'
}
