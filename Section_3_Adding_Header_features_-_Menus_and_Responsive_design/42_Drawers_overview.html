🗄️ Material UI Drawer Guide

📝 Study notes on how to use the <Drawer> component in Material UI.

❓ What Is a Drawer?

🗂️ The <Drawer> is a sliding panel anchored to one side of the screen (left, right, top, or bottom) that you can toggle to display navigation or content.

⬇️ How to Import

// Standard Drawer
import Drawer from '@material-ui/core/Drawer';

// Swipeable Drawer for mobile
import SwipeableDrawer from '@material-ui/core/SwipeableDrawer';

⚓ Key Props

🔹 anchor: where the drawer appears ('left', 'right', 'top', 'bottom').

🔹 open: boolean to control visibility.

🔹 onClose: function called to close the drawer.

💻 Basic Usage Example

<Drawer
  anchor="left"
  open={isOpen}
  onClose={() => setIsOpen(false)}
>
  {/* Content goes here */}
</Drawer>

📱 Swipeable Drawer

🔄 For mobile devices, use the swipeable version to enable swipe gestures for opening/closing:

<SwipeableDrawer
  anchor="left"
  open={isOpen}
  onOpen={() => setIsOpen(true)}
  onClose={() => setIsOpen(false)}
>
  {/* Navigation items */}
</SwipeableDrawer>

🗂️ Drawer Variants

📝 Temporary: sits above content and closes on click-away.

📌 Persistent: remains open until toggled.

🔳 Mini Variant: shows icons only, expandable on hover or click.

📋 Permanent: always visible, typically used on desktop layouts.
