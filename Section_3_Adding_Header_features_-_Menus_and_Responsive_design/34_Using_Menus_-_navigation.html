üîó Turning Menu Items into Links

To make each MenuItem navigate to a different page, we use the component prop and set it to Link from react-router-dom. We also use the to prop to define the route.

Example:

<MenuItem component={Link} to="/customsoftware">Custom Software Development</MenuItem>

‚ö†Ô∏è Gotcha: Active Tab Not Updating

Even though the content and URL update, the active tab doesn't change because we didn't update the state. To fix this, we update the value state when a menu item is clicked.

We use an arrow function:

onClick={() => {
  handleClose();
  setValue(1);
}}

‚úÖ Fixing the Navigation to Services Page

We also want a way to navigate directly to the /services page. We add another MenuItem at the top for this purpose:

<MenuItem component={Link} to="/services" onClick={() => { handleClose(); setValue(1); }}>Services</MenuItem>

üí° Summary

  • Use component={Link} and to prop for routing with MenuItem.
  • Update active tab with setValue() in onClick to reflect current tab.
  • Add a separate menu item for the main Services page to improve navigation.
