📘 Material UI Menu on Services Tab

🎯 Goal

🛠️ Add a pop-up menu to the “Services” tab using Material UI’s Menu component. On hover, display links to:

  • 🔹 Custom Software Development
  • 🔹 Mobile App Development
  • 🔹 Website Development

⚙️ Setup

  • 📥 Import Menu and MenuItem from Material UI.
  • 🔧 Use React’s useState for anchorEl and open/close state.
  • ➡️ Create handleClick to open the menu via event target.
  • ❌ Create handleClose to close the menu.

🔍 How It Works

  • 🖱️ On hover (or click) of the Services tab, call handleClick via onMouseOver (or onClick).
  • 📌 The menu is anchored to the tab via anchorEl.
  • 👋 Use MenuListProps={{ onMouseLeave: handleClose }} to close when the mouse leaves.

💡 Bonus Tips

  • 🔐 Add aria-owns and aria-haspopup props for accessibility.
  • 🆔 Link the menu ID to the tab via id props.
  • ✨ Material UI auto-styles MenuItem hover and selected states.

✅ Result

✅ Hovering over Services shows a clean pop-up menu of subpages. The menu closes on mouse leave, and the Services tab remains visible.
