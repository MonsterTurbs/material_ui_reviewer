📘 Material UI Menu Styling - Study Notes

🎨 1. Change the Menu Background

🔹 Use the Paper component inside the Menu to set a custom background color.

  • 💡 Create a style class: menu
  • 🎨 Use theme.palette.common.blue for the background
  • 🛠️ Apply with classes={{ paper: classes.menu }}

🧩 2. Style Menu Items Like Tabs

🔹 Keep menu items looking consistent with your tabs:

  • 💡 Create a class: menuItem
  • 📋 Apply theme.typography.tab styles
  • 🛠️ Use on MenuItems: classes={{ root: classes.menuItem }}

🖱️ 3. Hover & Opacity Effects

🔹 Make menu items appear like inactive tabs:

  • 🌑 Default opacity: 0.7
  • ☀️ On hover: opacity 1

✨ 4. Remove Shadow & Rounded Corners

  • ❌ No shadow: elevation={0}
  • 🔲 No rounded corners: borderRadius: 0

🎉 Gives the header a cleaner, more professional look.
