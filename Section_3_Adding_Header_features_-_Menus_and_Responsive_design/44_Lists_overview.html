📋 Material UI List Component

Simple and beginner-friendly guide to using the List component in Material UI.

📋 What is List?

The <List> component displays a vertical list of items—each item can include icons, text, or actions.

📥 How to Import

import List from '@material-ui/core/List';
import ListItem from '@material-ui/core/ListItem';
import ListItemIcon from '@material-ui/core/ListItemIcon';
import ListItemText from '@material-ui/core/ListItemText';

🏗️ Basic List Structure

A simple list example:

<List>
  <ListItem button>
    <ListItemText primary="Inbox" />
  </ListItem>
</List>

🎨 Adding Icons

Place an icon before the text:

<ListItem button>
  <ListItemIcon>
    <InboxIcon />
  </ListItemIcon>
  <ListItemText primary="Inbox" />
</ListItem>

🔎 Dense Variant

Use the dense prop to reduce spacing:

<List dense> ... </List>

🔘 Secondary Action

Add an action (e.g., delete button) on the right:

<ListItem
  secondaryAction={
    <IconButton edge="end" onClick={handleDelete}>
      <DeleteIcon />
    </IconButton>
  }
>
  ...
</ListItem>

✅ Selected Item

Track which item is selected using the selected prop:

<ListItem selected={activeIndex === i}> ... </ListItem>

📂 Nested List

Use <Collapse> for nested items:

<ListItem button onClick={toggleOpen}>Parent</ListItem>
<Collapse in={open}>
  <List disablePadding>
    <ListItem button className={classes.nested}>Child</ListItem>
  </List>
</Collapse>

👤 Avatars in List

You can also use an Avatar inside <ListItemIcon>:

<ListItemIcon>
  <Avatar>A</Avatar>
</ListItemIcon>

↪️ Inset List Text

Add left indentation using the inset prop:

<ListItemText inset primary="Nested Item" />

🪟 Virtualized List

For large lists, use react-window with <List> for virtualization.
