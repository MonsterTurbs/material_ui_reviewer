🔧 Integrating Responsiveness with Material UI

📚 Study notes for implementing responsive design using Material UI hooks and breakpoints.

📦 1. Import Hooks & Theme

➡️ Import useMediaQuery and useTheme from Material UI Core:

import useMediaQuery from '@material-ui/core/useMediaQuery';
import { useTheme } from '@material-ui/core/styles';

🛠️ Set up the theme in your component:

const theme = useTheme();

📺 2. Detect Screen Size

🔍 Create a boolean matches using a breakpoint:

const matches = useMediaQuery(theme.breakpoints.down('md'));  // true if screen is medium or smaller

🌿 3. Conditional Rendering

✂️ Use a ternary in JSX to show tabs only on larger screens:

{'{'}matches ? null : (
  <Tabs ... />
){'}'}

🚫 The output is null for medium and below, hiding the tabs.

↔️ 4. Responsive Styles

🎨 In your styles object, use theme.breakpoints.down() to adjust the logo height:

logo: {
  height: '8em',
  [theme.breakpoints.down('md')]: { height: '7em' },
  [theme.breakpoints.down('xs')]: { height: '5.5em' }
}

↕️ 5. Adjust Spacing

🧹 Include breakpoints for toolbar margin to remove extra gaps:

toolbarMargin: {
  marginBottom: '3em',
  [theme.breakpoints.down('md')]: { marginBottom: '2em' },
  [theme.breakpoints.down('xs')]: { marginBottom: '1.25em' }
}
