📘 Selected Menu Item - Material UI Study Notes

🧠 1. Use State to Track the Selected Menu Item

🔹 Create state to remember which menu option is active:

const [selectedIndex, setSelectedIndex] = useState(0);

📋 2. List Your Menu Options

🔹 Store your menu items in an array for easy rendering and indexing:

const menuOptions = [
  { name: 'Services', link: '/services' },
  { name: 'Custom Software', link: '/custom-software' },
  // ... add more options
];

🔁 3. Loop with .map()

🔹 Use menuOptions.map((option, index) => ...) to render each MenuItem:

  • 💡 option.name for the display text
  • 🔗 option.link for the route
  • 🔑 key={option.name} as React key

🖱️ 4. Attach an onClick Handler

🔹 Implement handleMenuItemClick(event, index):

  • ➡️ setAnchorEl(null)
  • ➡️ setOpen(false)
  • ➡️ setSelectedIndex(index)

🎨 5. Add selected Styling

🔹 Highlight the active item:

selected={index === selectedIndex && value === 1}

🔸 This applies when the Services tab (value = 1) is open.

🔗 6. Integrate React Router

🔹 Make each MenuItem a link:

  • 🔗 component={Link}
  • 🔗 to={option.link}

✅ 7. Results

  • 🎉 The selected menu item is highlighted.
  • 🎉 You can choose a service from the dropdown.
  • 🎉 Styling stays consistent with Material UI design.

💡 8. Bonus: Clean Code Tips

  • ✏️ Use an array for DRY rendering.
  • 🎨 Move reusable styles into the theme.
  • 🔑 Always add a unique key in loops.
