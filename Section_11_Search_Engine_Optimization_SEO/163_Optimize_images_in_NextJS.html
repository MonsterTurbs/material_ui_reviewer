🔌Install Image Optimizer

• Run npm install next-optimized-images imagemin-mozjpeg imagemin-optipng next-compose-plugins to add automatic JPEG/PNG compression. 😊

⚙️Configure next.config.js

• Wrap your exports using withPlugins to include withOptimizedImages. 🤓

• Example: 

const withPlugins = require('next-compose-plugins');
const withOptimizedImages = require('next-optimized-images');

module.exports = withPlugins([
  [withOptimizedImages, { /* your options */ }]
]);

🚀Restart Dev Server

• Run npm run dev to see changes. Fix any errors in the terminal.

🐞Fix _document.js Theme Import

• Correct theme path and naming:

import theme from '../src/UI/theme'; and use theme.palette.primary.main.

📦Install Missing Packages

• If you see react-lottie or Material‑UI errors, install them:

npm install react-lottie @material-ui/core @material-ui/styles

🔧Fix CSS url() Usage

• Wrap background URLs in backticks: background: `url('/assets/bg.jpg')`;.

✅Verify Pre-Rendering

• Open http://localhost:3000, check the Next.js badge at bottom-right in dev mode.

• This confirms server‑side rendered HTML is sent before React hydrates. ✨

🎉Success!

• You’ve migrated to Next.js with optimized images and SSR for better performance & SEO. 🎊
