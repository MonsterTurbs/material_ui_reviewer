📦Download Boilerplate

• Go to Material UI docs → Guides → “Server Side Rendering” → Next.js example.

• Run the given curl … command to grab the starter code.

⚙️Install & Run

• Inside the next.js folder, do:

– npm install

– npm run dev (opens at localhost:3000)

🗂️File-System Routing

• Place pages in /pages:

– index.js → “/”

– about.js → “/about”

• Add doctor.js → “/doctor” automatically.

🔗Link Component

• Use the boilerplate’s src/Link.js in place of React Router’s.

• Syntax: <Link href="/path">…</Link>

🎨Copy UI & Theme

• Copy your old UI/ folder into src/UI/.

• Delete the boilerplate’s theme, update imports in pages/_app.js to UI/theme.js.

📂Static Assets (public/)

• Create public/assets/ for images and SVGs.

• Refer to them as /assets/image.svg in <img> tags—no JS import needed.

✏️Rename Files

• Remove spaces: e.g. custom software icon.svg → custom-software.svg.

🗄️Copy Assets Folder

• Paste your old assets/ into public/assets/.

🎞️Animations Folder

• Copy your animations/ (JS/JSON files) into src/animations/.

• Don’t put JS here in public/; keep them processed by Webpack.

🔄Update Imports

• In your pages, update all imports of images to "/assets/…".

• Update component imports: e.g. src/UI/ButtonArrow instead of old paths.

✅Pages to Copy

• Copy all your route components (about, contact, estimate, etc.) into /pages.

• Rename your landingPage.js → index.js for the home route.
