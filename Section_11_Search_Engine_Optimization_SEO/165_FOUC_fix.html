🖋️ Adding Fonts & Favicon

• Copy Google Fonts link and favicon from your CRA index.html into _document.js in Next.js.

• Paste the <link> tags inside the <Head> section.

• Place favicon.png into Next.js /public and link to it.

🧹 Removing CssBaseline

• CssBaseline from Material UI resets styling, but if your CRA project didn’t use it, remove it.

• Delete the <CssBaseline/> import and JSX in _app.js.

🚫 Resetting Body Margin

• Browsers add default margin: 8px on <body>.

• In _document.js, add <body style="margin:0"> to remove the gap.

📦 Installing Axios

• Next.js doesn’t include Axios by default.

• Run npm install axios to add it for HTTP requests.

🐞 Fixing Header Infinite Loop

• In header.js, useEffect was setting state without guarding.

• Wrap the state-update in if (value !== newValue) so it only runs once.

⚙️ CSS url() Backticks

• CSS URLs in JS need backticks: backgroundImage: `url('/assets/bg.jpg')` or quotes.

• Without them, you get URL is not defined errors.

🔀 Avoiding FOUC in Header

• Next.js SSR doesn’t know screen size, so MUI media queries flash wrong layout.

• Wrap desktop tabs in <Hidden mdDown> and mobile drawer in <Hidden lgUp>.

✅ Verifying SSR with curl

• Run curl -A "Googlebot/2.1" http://localhost:3000 to see what bots get.

• A blank CRA shows only JS stub; Next.js returns full HTML pre-rendered.

📂 Public Folder for Static Assets

• Create /public for images and files.

• Access via /assets/logo.svg without JS imports, reducing bundle size.
