⚠️Important Next.js Fix

In the upcoming “Migrate from create-react-app to NextJS” lecture, you’ll use a Material-UI boilerplate wired up for Next.js.

🐞The Issue

The official boilerplate expects the latest Material-UI and Next.js versions, which currently aren’t compatible with our code.

⬇️What to Do

Instead of cloning the GitHub example, download the provided zip attached to this lecture. It has pin-matched dependency versions that work seamlessly with our project.

✅Why It Matters

  • Avoid version conflicts 🚫
  • Save time on troubleshooting 🕒
  • Ensure the boilerplate runs out-of-the-box 🏃‍♂️
