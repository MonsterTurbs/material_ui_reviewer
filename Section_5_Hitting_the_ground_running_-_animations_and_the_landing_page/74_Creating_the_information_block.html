🏁 Final Section: Info Block

This is the last roadmap piece on the home page. It guides you to the “About Us” and “Contact Us” pages! ℹ️

📐 Layout Structure

  • Outer grid item ➡️ holds the whole block with a background image.
  • Inside, a grid container with direction="row" splits into two columns:
    • Left column: About Us section.
    • Right column: Contact Us section.

🖼️ Background Image

We use a CSS class .info-background on a <div> to:

  • Set background-image: url(...) 🎨
  • Give it a fixed height: 80rem so it shows up properly.

📦 About Us Section

  • Grid item with position: absolute to sit on top of the image.
  • Inside: a column container with:
    • <Typography variant="h2">About Us</Typography> 📝
    • <Typography variant="subtitle2">Let’s get personal.</Typography> 💬
    • A “Learn More” button with white text and border 🔗
  • We add margin-left: 5rem and use alignItems="center" to center vertically.

📦 Contact Us Section

  • Same structure as About Us, but with contact text:
    • <Typography variant="h2">Contact Us</Typography> ✉️
    • <Typography variant="subtitle2">Say hello!</Typography> 👋
    • A white “Learn More” button 🔗
  • Also positioned absolutely and centered the same way.

⚙️ Positioning & Spacing

  • position: absolute on both columns to layer them over the background.
  • alignItems="center" to vertically center inside the row container.
  • margin-left adds padding from the edge so text isn’t flush.

🎨 Button Styling

  • Wrap each button in its own grid item so it doesn’t stretch full width.
  • Override styles:
    • Text color: white 🤍
    • Border color: white 🤍
  • Keep the same padding and size for consistency.
