🔧 Tweak Bodymovin JSON for React

🎯 Goal

Fix the exported .json so images are bundled in JavaScript, not referenced by URL. Even beginners can follow! 👶🚀

🗂️ 1. Locate Your Exported Files

Find the folder with Data.json and its images/ folder that Bodymovin created. 📁

✂️ 2. Copy JSON

Open Data.json in your editor, select all, and copy it. We’ll paste it into a new JS file. 📋

📝 3. Create data.js

On your desktop (or project folder), create data.js and start with:

export default {
  /* paste JSON here */
};

🖼️ 4. Import Images

At top of data.js, add imports for each image file:

import img0 from './images/00001.png';
import img1 from './images/00002.png';
// etc...

🔄 5. Replace URLs with Imports

  • In the JSON, find each "u":"..." (URL) and change to "u":"" (empty string).
  • Replace each "p":"00001.png" with the imported variable, e.g. p: img0.

💾 6. Final Export

Your data.js should now import images and export the JSON data object. No external URLs! ✅

🗑️ 7. Clean Up

Delete the old Data.json and /images in that folder; you now have data.js bundling everything. 🧹

🚀 8. Next Steps

Import data.js into your React component and use a Lottie player to render the animation! 🎉
