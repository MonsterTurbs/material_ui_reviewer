🆕 1. Create New Component File

Make a new file CallToAction.js in your UI folder. This will hold the “Free Estimate” section you’ll reuse on many pages.

🔧 2. Import What You Need

  • React for components
  • Grid and Typography from Material UI
  • Button and your custom ButtonArrow
  • makeStyles and useTheme for styling

🎨 3. Set Up Styles

  • Use makeStyles(theme => ({ ... })) to define CSS.
  • Grab theme inside your component via useTheme().
  • Leave the style object empty at first, then add rules as you build.

🏗️ 4. Build Component Structure

  • Return a <Grid container> (row by default).
  • First Grid item: a text block (itself a container direction="column").
  • Second Grid item: the “Free Estimate” button.

📝 5. Add Text Section

  • Inside the left block, use <Typography variant="h2"> for title.
  • Add a line break (<br/>) to split lines.
  • Use variant="subtitle2" below for smaller description.

🔘 6. Add & Style Button

  • Paste in the <Button variant="outlined">Learn More</Button> block.
  • Wrap arrow icon with <ButtonArrow fill={theme.palette.common.blue} />.
  • Tweak margin so text and arrow have space.

🌄 7. (Optional) Add Background Image

  • Place a <div> under your grid with className={classes.background}.
  • In CSS: background-image: url(...); background-size: cover;
  • Use position: absolute & appropriate height to overlay grid.

📐 8. Make It Responsive

  • Import useMediaQuery to detect small screens.
  • Switch layout to direction="column" at extra-small breakpoints.
  • Adjust paddings, font sizes, and background image for mobile.

✅ 9. Use It in Pages

  • Import your CallToAction into LandingPage.js.
  • Add it as a final <Grid item> under other sections.
  • Enjoy a consistent “Free Estimate” prompt everywhere! 🎉
