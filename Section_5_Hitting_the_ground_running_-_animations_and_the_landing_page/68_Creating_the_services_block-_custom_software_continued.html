⚙️ Spacing & Responsive Tips

Easy steps to make our Services section look great on any device! 📱💻

🖼️ 1. Add Icon Spacing

Give the icon some room from the text:

.icon {
  margin-left: theme.spacing(2);
}

— That adds a nice gap on larger screens.

📏 2. Remove on Mobile

On very small screens, we don’t want extra space when items stack:

[theme.breakpoints.down('xs')]: {
  icon: { margin-left: 0 }
}

↕️ 3. Space Between Sections

Push the services block down from the hero:

.serviceContainer { margin-top: theme.spacing(12); }

⬅️ 4. Left Margin for Text

Move the text/buttons inward:

.serviceItem { margin-left: theme.spacing(5); }

Remove on small screens:

[theme.breakpoints.down('sm')]: {
  serviceItem: { margin-left: 0 }
}

📱 5. Center on Small Screens

Use a media query in React to center items only on small devices:

const matchesSM = useMediaQuery(theme.breakpoints.down('sm'));
<Grid container justify={matchesSM ? 'center' : undefined}>...

🎨 6. Text Alignment

Also center text when stacked:

<Grid item
  textAlign={matchesSM ? 'center' : undefined}>...

↔️ 7. Button Spacing on Mobile

Add a bottom margin under the “Learn More” button when stacked:

[theme.breakpoints.down('sm')]: {
  learnButton: { marginBottom: theme.spacing(2) }
}

🔒 8. Prevent Edge Touch

Add padding around links on small screens so they never touch the edge:

[theme.breakpoints.down('sm')]: {
  serviceContainer: { padding: 25 }
}

✅ Result

  • 👍 Icons & text have breathing room
  • 🔄 Layout stays neat on small screens
  • 📏 Sections are nicely separated
  • 🎯 Perfect for both desktop & mobile!
