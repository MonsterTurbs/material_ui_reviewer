🏠 Landing Page Overview - Study Notes

🎯 Goal

Create a friendly, clear Home (landing) page that acts as a roadmap for the rest of the site and guides users to other pages. 🚀

🧩 Page Structure

  • 1. Hero Block – Top section with an animation and headline to grab attention. 🎬
  • 2. Services Block – Icons and short descriptions of services in a grid. 🛠️
  • 3. Revolution Block – Floating card with text and background image. 🔄
  • 4. Info Block – About & Contact sections side-by-side with a background image. ℹ️
  • 5. Call to Action – Banner encouraging free estimates, reused on many pages. 📞

🛠️ Animation Setup

We’ll export an animation (e.g., CPU passing hands) from After Effects, use a React animation library, then make it resize responsively. ✨

📐 Grid Layouts

Each block uses Material UI’s Grid container/item props. Think:

  • container wraps all items.
  • item defines each block or link.
  • Use direction, justify, alignItems, and spacing for layout.

Grid helps align content and keep pages consistent and responsive. 📱💻

🔀 Responsive Tips

  • Use Hidden to show/hide blocks by screen size (e.g., hide footer links on mobile). 👀🙈
  • Apply breakpoints (theme.breakpoints) in styles for fonts, margins, and sizes. 📏

✏️ Design Tweaks

Feel free to adjust alignment or sizes from the design file if it improves clarity—trust your eye! 👀🎨
