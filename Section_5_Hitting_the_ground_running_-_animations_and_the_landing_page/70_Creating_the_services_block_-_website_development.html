🌐 Adding Website Development Section

📥 1. Import the Website Icon

Bring in the new icon from your assets folder:

import WebsiteIcon from '../../assets/websiteIcon.svg';

📋 2. Copy the Template Block

Reuse your Custom Software block as a starting point:

// Select and copy the entire <Grid item>…</Grid> for Custom Software

📝 3. Paste & Rename

  • Paste it under the iOS / Android block.
  • Change the comment to Websites Block.
  • Update the heading text: Website Development.

✍️ 4. Update Descriptions

Change the two subtitle lines:

<Typography variant="subtitle1">Reach more, discover more, sell more</Typography>
<Typography variant="subtitle1">Optimized for search engines, built for speed</Typography>

🖼️ 5. Swap in the New Icon

Update the image tag:

<img src={WebsiteIcon} alt="Website icon" />

💾 6. Save & Verify

🔄 After saving, check that the new section appears correctly on desktop:

  • The Websites block sits on the right side.
  • Text and button on the left match the template.

📱 7. Test Responsiveness

Shrink the browser to mobile size:

  • Ensure all sections stack vertically and center.
  • Icons and text remain legible and spaced nicely.

✅ Result

  • ✅ Website Development section added!
  • 📐 Looks balanced on both desktop and mobile.
  • 🎨 Consistent styling with other service blocks.
