🛠️ Button Wiring & Cleanup

Easy-to-follow steps for hooking up buttons and fixing small bugs.

🔗 1. Hook up Header “Free Estimate”

In Header.js, add to the Estimate button: component={Link} to="/estimate" and then onClick={() => { props.setValue(5); }} to clear the active tab. Also update your refresh-fix useEffect switch to handle '/estimate'.

🏠 2. Wire Landing-Page Buttons

In LandingPage.js import: import { Link } from 'react-router-dom';

Then for each button (Hero, Services, Revolution, Info, CTA), add:
component={Link} to="/your-route"
e.g. Learn More → to="/revolution", Estimate → to="/estimate".

📥 3. Pass State Props to Landing Page

In App.js, routes like:
<Route path="/" component={LandingPage} />
don’t accept props. Change to: render={props => <LandingPage {...props} setValue={setValue} setSelectedIndex={setSelectedIndex} />}

🐞 4. Quick Bug Fix: Routing & Active Tabs

After wiring, test each button. If clicking a button doesn’t update the tab, ensure you call
props.setValue(index) in its onClick (e.g. 5 for Estimate). And handle that route in your header’s useEffect.

Happy coding! 🚀
