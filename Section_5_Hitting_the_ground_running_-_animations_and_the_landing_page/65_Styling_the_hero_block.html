📚 Hero Block Styling Guide

⚙️ Align Animation Right

Use the justifyContent="flex-end" prop on the Grid container (direction="row") to push the animation all the way to the right edge.

<Grid container justifyContent="flex-end" …>

↕️ Center Text Vertically

To center the text and buttons with the animation, add alignItems="center" on the same Grid container.

<Grid container alignItems="center" …>

✍️ Use Typography for Text

Replace a plain <div> with Material UI’s <Typography> component for better styling. Center text with align="center" and make it large with variant="h2".

<Typography variant="h2" align="center">Your Slogan</Typography>

🎨 Customize H2 in Theme

  • fontFamily: 'Raleway'
  • fontWeight: 700
  • fontSize: 2.5rem
  • color: theme.palette.primary.main
  • lineHeight: 1.5

📐 Responsive Layout with Grid

Give each Grid item the sm prop (no number) to auto-size horizontally until below small screens, where they stack vertically:

<Grid item sm>…</Grid>

📏 Constrain Animation Size

Apply a CSS class to the animation wrapper to limit its size:

.animation {
  max-width: 50rem;
  min-width: 21rem;
  margin-top: 2rem;
  margin-left: 10%;
}

🔀 REM (M) vs Percentage (%)

  • 🌟 REM (e.g. 2rem) gives a constant gap across devices.
  • 📊 Percentage (e.g. 10%) scales the gap with screen width.
