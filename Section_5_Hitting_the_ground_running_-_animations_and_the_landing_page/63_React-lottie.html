🎬 React Lottie – Study Notes

🎯 Goal

Learn how to render Bodymovin JSON animations in React using React Lottie. Even beginners can do it! 👶✨

📦 1. Install Package

Run in terminal:

npm install --save react-lottie

⚙️ 2. Set Up Default Options

Create an options object:

const defaultOptions = {
  loop: true,           // 🔁 repeat animation
  autoplay: true,       // ▶️ start automatically
  animationData: animationData,  // imported JSON
  rendererSettings: {
    preserveAspectRatio: 'xMidYMid slice'
  }
};

🚀 3. Render Animation

Import and use the <Lottie /> component:

import Lottie from 'react-lottie';
import animationData from '../animations/landing/data.js';

// inside render:
<Lottie
  options={defaultOptions}
  height="100%"
  width="100%"
/>

🛠️ 4. Integrate in App

  • Replace dummy component in App.js route "/" with your new <LandingPage />.
  • Ensure you import your LandingPage correctly.

🧐 5. Inspect SVG

The animation is rendered as SVG, giving you crisp, scalable graphics! 🖼️👍

💡 6. Tips

  • You can use fixed pixel values or percentages for height/`width`.
  • Toggle loop or autoplay as needed for different effects.

⚠️ 7. Note

React Lottie may show a deprecation warning for old lifecycle methods. It still works, but future updates may fix this. ⏳
