🚀 Adding iOS / Android App Section

🔤 1. Rename Menu Item

Change “Mobile App Development” to “iOS / Android App Development”:

// In Header.js menu options: { name: "iOS / Android App Development", link: "/mobile-apps" }

Also update in the footer links the same way.

📋 2. Copy Custom Software Block

Use the Custom Software section as a template:

  • Select the entire <Grid item>…</Grid> for Custom Software.
  • Paste it right beneath to create the new section.

📥 3. Import the Mobile Icon

Add at the top:

import MobileIcon from '../../assets/mobileIcon.svg';

✍️ 4. Update Text

Change the headings and descriptions:

<Typography variant="h4">iOS / Android App Development</Typography>
<Typography variant="subtitle1">Extend functionality…</Typography>
<Typography variant="subtitle1">Integrate your web experience…</Typography>

🔀 5. Conditional Line Break

Only on larger screens, insert a line break:

{/* inside JSX */} {"smallScreen ? null :
}

— ensures the text wraps nicely on desktop but stays on one line on mobile.

➡️ 6. Align Section Right / Center

Push the block to the right on desktop, center on mobile:

<Grid container
  justify={matchesSM ? 'center' : 'flex-end'}
>…</Grid>

Use useMediaQuery(theme.breakpoints.down('sm')) to set matchesSM.

🖼️ 7. Update Icon & Alt Text

Swap in the new icon:

<img src={MobileIcon}
  alt="Mobile phone icon"
/>

↔️ 8. Add Right Margin

Give space on the right to match the left side:

{/* on the icon’s wrapper Grid item: */} style={{ marginRight: theme.spacing(5) }}

✅ Result

  • 📱 Section title changed to iOS / Android
  • ✅ New block appears on the right on desktop
  • ↕️ Centers gracefully on mobile
  • 👍 Balanced spacing on both sides

All set! 🎉
