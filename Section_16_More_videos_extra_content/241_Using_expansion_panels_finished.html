🎯 Main Goal

• Show how to style the “Services” expansion panel tab as “selected” when on the Services page. ✅

• Use Material UI’s makeStyles hook to read React props inside styles! 🎉

🔑 Passing Props into useStyles

1. In your component, gather props (e.g. value for current tab index).

2. Call your hook with those props:
const classes = useStyles(props); 🙌

3. Inside makeStyles, receive them:

makeStyles((theme) => (props) => ({ /* …styles… */ })) or simply destructure ({ value }). 🔧

🎨 Conditionally Styling the Summary

• To darken “Services” when value === 1:

expansionSummary: {
  backgroundColor: props => props.value === 1
    ? 'rgba(0,0,0,0.14)'
    : 'inherit',
},

• “Inherit” keeps the normal look when not selected. 👍

💡 Highlighting the Text

• Brighten the label text only if selected:

listItemText: {
  opacity: props => props.value === 1 ? 1 : undefined,
},

• Full opacity (1) makes it stand out! ✨

🛠️ Small Tweaks

• Add hover effect on summary:

'&:hover': {
  backgroundColor: 'rgba(0,0,0,0.08)',
