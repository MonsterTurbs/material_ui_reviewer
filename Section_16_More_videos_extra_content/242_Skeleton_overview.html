🔍 What is a Skeleton?

• A Skeleton is a placeholder shape (circle, rectangle, or line) shown while data is loading 😊

• It helps users see where content will appear instead of staring at a spinner 🔄

📚 Variants & Shapes

• variant="text": shows a line, like where a paragraph will go 📝

• variant="rect": shows a rectangle, like an image or card 📦

• variant="circle": shows a circle, like an avatar or icon ⚪️

✨ Animation Options

• animation="pulse" (default) makes the skeleton softly fade in/out 🔄

• animation="wave" creates a left-to-right shimmer 🌊

• animation={false} disables animation for a static grey placeholder 🚫

💡 Practical Examples

• YouTube Preview: Rectangle for video thumbnail + text lines for title & details ▶️

• Social Post: Circle for avatar + several text lines for post content 👤📰

🔧 How to Use

1. Import from the lab package:

import Skeleton from '@mui/lab/Skeleton';

2. Track loading state:

const [loading, setLoading] = useState(true);

3. Conditionally render:


{loading ? (
  <Skeleton variant="rect" width={210} height={118} animation="wave" />
) : (
  <img src="..." alt="Loaded" />
)}

⚡️ Benefits

• Makes UI feel faster and more polished 🚀

• Guides the eye to where content will appear 👀

• Reduces “loading spinner anxiety” by showing real layouts 🕗

👍 Tips

• Match skeleton shapes/sizes to real content for best effect 🎨

• Keep animation subtle—don’t distract from loading flow ✨

• Use in lists, cards, avatars, text blocks—wherever data is delayed ⏳
