üìå Introduction to Inline Styling

Inline styling is used when you want to apply custom styles to a specific component instead of using the theme. Material UI makes this easy using makeStyles.

üõ†Ô∏è How It Works

  • Import makeStyles from @material-ui/styles
  • Create a style hook using JavaScript object-style (camelCase props like borderRadius)
  • Call the hook inside your component to access classes
  • Use className={classes.myStyle} to apply
const useStyles = makeStyles({
  myButton: {
    backgroundColor: "blue",
    color: "white",
    padding: "10px"
  }
});

const MyComponent = () => {
  const classes = useStyles();
  return <button className={classes.myButton}>Click Me</button>
}

üß† Bonus: Access Theme Inside Styles

You can pass a function to makeStyles to access the theme object:

const useStyles = makeStyles((theme) => ({
  root: {
    padding: theme.spacing(2),
    marginTop: theme.mixins.toolbar.minHeight
  }
}));

‚úÖ Real Use: Fixing Header Margin

  • The AppBar is fixed, so it overlays content
  • Use theme.mixins.toolbar to apply a margin/padding spacer
  • This avoids overlap and is the recommended Material UI fix

üì¶ Summary

  • Use inline styling with makeStyles when you want component-level styles
  • Use theme object to ensure consistency with Material UI design
  • Combine inline styles and themes for powerful design control
