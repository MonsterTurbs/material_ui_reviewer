Material UI Tabs - Study Notes

🔍 What are Tabs?

The Tabs component is used to navigate between different sections or pages of your app.

You'll see it in the header as clickable links.

⚙️ Components

Tabs - the container holding all tab items.

Tab - each individual tab item.

label - the text displayed on each tab.

🧠 State Management

Use React's useState hook to track the active tab.

value - the currently selected tab index.

onChange - the event handler called when a tab is clicked.

💡 How Does It Work?

When you click a tab, the onChange function updates the value. That change activates the new tab.

📌 Takeaway

The Tabs component is a simple yet powerful tool in Material UI.

It works seamlessly with React Router for smooth navigation in your app.
