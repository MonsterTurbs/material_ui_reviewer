üñåÔ∏è Goal

Customize the colors in your React app using Material UI's palette object in the theme. We'll also connect our app to use those colors globally.

üìã Steps to Customize Palette

  1. Open theme.js in your UI folder.
  2. Create variables for your main brand colors.
  3. Use those variables in the palette object.
  4. Material UI will auto-generate light and dark shades.

üíª Sample Code

const arcBlue = "#0B72B9";
const arcOrange = "#FFBA60";

export default createMuiTheme({
  palette: {
    common: {
      blue: `${arcBlue}`,
      orange: `${arcOrange}`
    },
    primary: {
      main: `${arcBlue}`
    },
    secondary: {
      main: `${arcOrange}`
    }
  }
});

üéØ Why This Matters

  • You can reference theme.palette.primary.main anywhere in your components.
  • You only need to update the color in one place (inside theme.js).
  • Improves maintainability and consistency of your UI.

üîÑ Applying to Components

Example using AppBar:

<AppBar color="primary"> // uses theme.palette.primary.main
// or
<AppBar color="secondary"> // uses theme.palette.secondary.main

‚úÖ Final Result

Once connected, your AppBar or any Material UI component using theme colors will automatically reflect your custom palette.
