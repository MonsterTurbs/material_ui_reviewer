üìö What is Typography in Material UI?

Typography controls all the text styles in your app like headings, body text, captions, buttons, etc. Material UI provides default variants like h1 to h6, subtitle1, body1, caption, and more.

‚öôÔ∏è Where to Customize It

Inside your theme.js file, add a new object called typography:

export default createMuiTheme({
  typography: {
    h3: {
      fontWeight: 300
    }
  }
});

üñãÔ∏è How to Use Typography in Components

Use the <Typography> component with the variant and color props:

import Typography from '@material-ui/core/Typography';

<Typography variant="h3" color="secondary">
  Arc Development
</Typography>

üß† Notes on Styling

  • variant="h3" uses styles from theme.typography.h3
  • color="primary" or color="secondary" pulls from theme.palette
  • You can also define fontFamily, lineHeight, etc. in the theme
