🎯 Goal

🏷️ Add an orange "Free Estimate" button to the top-right of the header, matching the design mockup.

🔢 Step 1: Import the Button

📂 At the top of Header.js, import Material UI's Button component:

import Button from '@material-ui/core/Button';

➕ Step 2: Add the Button

⚙️ Place the button inside the <Toolbar>, just below the tabs:

<Button variant="contained" color="secondary">Free Estimate</Button>

🎨 Step 3: Style the Button

🖌️ In your styles object, create a button class with:

  • 🔸 borderRadius: '50px'
  • 🔸 marginLeft: '50px' and marginRight: '25px'
  • 🔸 fontFamily: 'Pacifico', fontSize: '1rem', textTransform: 'none'
  • 🔸 height: '45px', color: 'white'

📦 Step 4: Move Styles to Theme

📦 In theme.js, add these under typography.estimate:

estimate: {
  fontFamily: 'Pacifico',
  fontSize: '1rem',
  textTransform: 'none',
  color: 'white'
}

👉 Then use ...theme.typography.estimate in your button style to inherit.

✅ Success!

🎉 You now have a beautifully styled "Free Estimate" button in your header. Well done!
