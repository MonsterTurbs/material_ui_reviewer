🧱 Study Notes: Creating the Header Component

📌 Why Start with the Header?

  • The header shows on every page of the website
  • Acts like the "backbone" or "top navigation"
  • Gives us a consistent layout structure before adding other content

🔧 What We'll Use

  • AppBar – Material UI component for headers
  • Toolbar – Used inside AppBar to layout items horizontally

📁 Folder Structure Setup

  • Create src/UI folder
  • Create src/components folder
  • Move App.js inside components
  • Update the import in index.js to: ./components/App.js
  • Create Header.js inside UI

📄 Code for Header.js

import React from "react";
import AppBar from "@material-ui/core/AppBar";
import Toolbar from "@material-ui/core/Toolbar";

export default function Header(props) {
  return (
    <AppBar position="fixed">
      <Toolbar>Arc Development</Toolbar>
    </AppBar>
  );
}

🔁 Use It in App.js

  • Import the header:
    • import Header from './UI/Header'
  • Add it before the content:
    • <Header />

📣 What's Happening?

  • <AppBar /> makes a full-width top bar
  • <Toolbar /> makes the content inside align horizontally
  • Using position="fixed" keeps it always at the top
  • If you change it to static, the header shrinks and scrolls with the page

✅ Summary

  • We've built a reusable header using Material UI
  • It spans full width and stays at the top
  • We're ready to customize it further later on
