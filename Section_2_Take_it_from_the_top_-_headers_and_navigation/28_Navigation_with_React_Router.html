📚 React Router Navigation - Study Notes (Beginner Friendly)

🔧 Setting Up React Router

To enable navigation in our React app, we use React Router DOM.

import { BrowserRouter, Route, Switch } from 'react-router-dom'

  • BrowserRouter - handles URL changes
  • Switch - renders only the first matching route
  • Route - renders component based on path

🧱 Route Setup

Inside App.js:

<BrowserRouter>
  <Header />
  <Switch>
    <Route exact path="/" component={() => <div>Home</div>} />
    <Route path="/services" component={() => <div>Services</div>} />
    ... (other routes)
  </Switch>
</BrowserRouter>

This makes each page render content based on the URL.

🧭 Linking Tabs to Pages

To make header tabs navigate to pages:

import { Link } from 'react-router-dom'

<Tab component={Link} to="/services" label="Services" />

Each Tab uses component=Link and to= props to navigate.
