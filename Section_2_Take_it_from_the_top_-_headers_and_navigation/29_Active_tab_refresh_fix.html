üìå Fixing Active Tab Issue on Page Refresh

  • Problem: When the page is refreshed, the tab always highlights "Home" even if we're on another route like "/services".
  • Solution: Use useEffect to detect the current URL and update the active tab accordingly.
  • Check window.location.pathname and call setValue based on the current path.
  • Add value to the dependency array of useEffect to avoid infinite updates.

üîÅ Example useEffect Setup

useEffect(() => {
  if (window.location.pathname === "/" && value !== 0) setValue(0);
  else if (window.location.pathname === "/services" && value !== 1) setValue(1);
  ...
}, [value]);

  • Wrap the logo <img> in a Material UI <Button>.
  • Add component={Link} and to="/" to make it a link.
  • Add onClick={() => setValue(0)} to update the active tab when clicked.
