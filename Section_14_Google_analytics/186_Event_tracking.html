🎯 What Are Analytics Events?

Events track specific user actions—like clicking buttons—so you know what users do on your site. 👆

🧩 Event Properties

  • Category: Group name (e.g., "estimate").
  • Action: Description (e.g., "desktop header pressed").
  • Label (optional): Extra detail (e.g., button text).
  • Value (optional): A number (e.g., purchase amount).

🛠️ Tracking Buttons in React

Use react-ga to fire an event on click:

import ReactGA from 'react-ga';

// Example: desktop header "Estimate" button

📋 Buttons to Track

  • 🏠 Home page "Estimate" ➡️ home page pressed
  • 🖥️ Desktop header ➡️ desktop header pressed
  • 📱 Mobile header ➡️ mobile header pressed
  • 🎨 Call-to-action section ➡️ call to action pressed
  • ✉️ Contact page button ➡️ contact page pressed

🚀 Tracking Form & Estimate Events

  • 📨 Message sent ➡️ ReactGA.event({category:'message',action:'sent message'})
  • 🔍 Estimate checked ➡️ fire when "Get Estimate" opens dialog
  • ✅ Estimate sent ➡️ estimate sent after submission

🔍 Verify in Google Analytics

  • Go to Real-Time > Events in GA dashboard.
  • Click buttons and watch your events appear live! 👀

✅ Quick Recap

  • 📦 Install react-ga and initialize with your Tracking ID.
  • 🎯 Choose categories & actions for each button.
  • 🔧 Add ReactGA.event(...) in onClick handlers.
