🛠️ Step 1: Go to Google Analytics

  • 🔍 Visit analytics.google.com
  • 🔐 Sign in with your Google account (or create one).
  • 🖱️ Click “Start measuring.”

📝 Step 2: Create Your Account & Property

  • 📛 Account Name: e.g., Material-UI-Course
  • 🌐 Property (Website) Name: e.g., Arc Development
  • 🔗 Enter your site URL (use http or https).
  • 🌍 Select industry category and time zone.

🔢 Step 3: Get Your Tracking ID

After setup, copy the Tracking ID (e.g., G-XXXXXXXXXX).

🔍 Step 4: Explore Analytics Dashboard

  • 📊 Home: Page views, active users, top pages.
  • ⏱️ Real-Time: Live user count & page views.
  • 🌍 Audience: Who your users are (location, device).
  • 🚀 Acquisition: Where users come from (search, social).
  • 🔧 Behavior: What users do (page views, events).
  • 🏆 Conversions: Track goals (e.g., “Estimate requested”).

✨ Step 5: Install & Initialize in Next.js

  • 📦 Install package: npm install react-ga
  • 📄 In _app.js:
  • ```js import ReactGA from 'react-ga'; ReactGA.initialize('G-XXXXXXXXXX'); ```
