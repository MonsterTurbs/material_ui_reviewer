1. Create Firebase Project 🔥

• Go to firebase.google.com and sign in with your Google account.
• Click Create a project, name it “Material UI Course”, disable Analytics, and finish setup. 🎉

2. Install Firebase CLI 🛠️

Open your terminal and run:

npm install -g firebase-tools

This gives you the firebase command anywhere. 🚀

3. Log In to Firebase 🪪

In terminal, run:

firebase login

• Allow anonymous error reporting.
• A browser window opens—pick your Google account and grant permissions.
• Back in terminal you’ll see “success logged in as …”. ✅

4. Initialize Functions 🚧

In your project folder:

firebase init

• Select Functions feature.
• Choose your existing “Material UI Course” project.
• Pick JavaScript and enable ESLint.
• Agree to install dependencies. 📦

5. Explore the Functions Folder 📂

• A new functions/ folder appears.
• Open functions/index.js—it has initial boilerplate and deploy instructions. 👀

6. Configure Firebase Admin & Functions ⚙️

In functions/index.js, add:

const functions = require('firebase-functions');
const admin = require('firebase-admin');
const config = functions.config();
admin.initializeApp();

This links your code to Firebase services. 🔗

7. Ready to Deploy & Extend 🚀

  • Next: install and configure NodeMailer inside functions/.
  • Write your email-sending function.
  • Deploy with firebase deploy --only functions.
  • Call your Cloud Function URL from the front-end to send emails. ✉️
