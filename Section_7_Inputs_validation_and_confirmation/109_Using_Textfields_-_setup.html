1. Create & Wire Up Contact.js 📁

• In components, add Contact.js.
• In App.js, import Contact from './components/Contact';
• Replace dummy route with <Route path="/contact" render={() => <Contact />} />.
Now the page is ready to load! 🚀

2. Import Core Tools 📥

At the top of Contact.js, import:
import React from 'react';
import { Link } from 'react-router-dom';
import { makeStyles, useTheme } from '@material-ui/core/styles';
import Grid from '@material-ui/core/Grid';
import Typography from '@material-ui/core/Typography';
import Button from '@material-ui/core/Button';
import TextField from '@material-ui/core/TextField';
These give you layout, text, buttons, and form inputs. 🛠️

3. Setup Styles & Theme 🎨

• Define useStyles = makeStyles(theme => ({ background: { ... } })).
• In the Contact function, get classes = useStyles() and theme = useTheme().
• Return <Grid container direction="row" spacing={4}> to start the layout. 😊

4. Add Left Column Container 🔲

• Wrap the form area:
<Grid item>
  <Grid container direction="column" spacing={2}>
   ...form & info here...
  </Grid>
</Grid>
This lets us use alignItems or justify on the wrapper later. 🎯

5. Add Title & Intro Text 🏷️

Inside the column container, add:
<Grid item><Typography variant="h2">Contact Us</Typography></Grid>
<Grid item><Typography variant="body1" paragraph style={{ color: theme.palette.common.blue, lineHeight:1 }}>Have questions? We’re here to help!</Typography></Grid>
Set text color and line height. ✏️

6. Import & Style Background 🌄

• Import image:
import background from '../assets/background.jpg';
• Copy styles from Landing page:
background: { backgroundImage: `url(${background})`, backgroundSize:'cover', height:'60m' } in useStyles.
• Later apply class classes.background to right column. 🎨

7. Add Background Column & Split Grid 🔀

• After left Grid item, add:
<Grid item className={classes.background}></Grid>
• Use lg={3} on left and lg={9} on background to split 3/9 columns.
• Now form uses 1/4 width, background uses 3/4. 📐

8. Preview & Adjust 🔍

Save and refresh. The text and background should share the row space correctly (3:9 ratio). Resize to confirm it stays balanced. 👍

9. Next Steps ➡️

Now that structure is set, next we’ll build the actual form fields, apply validation, and make everything functional. Stay tuned! 🚀
