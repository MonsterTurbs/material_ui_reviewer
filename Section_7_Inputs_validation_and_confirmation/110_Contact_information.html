1. Container Direction: Column 📐

When you set direction="column", the main axis is vertical. Using justify="center" will center your items from top to bottom. 😊

2. Importing Icons 📱✉️

Under your other imports, add: import phoneIcon from "assets/phone.svg"; and import emailIcon from "assets/email.svg";

3. Adding Icon & Text Together 🔗

Inside your grid with direction="column", add a new grid item container (default row):

  • Wrap the <img src={phoneIcon} alt="phone"/> in a <Grid item>.
  • Next to it, add a <Typography variant="body1">555-555-5555</Typography>.

4. Spacing Between Icon & Text ↔️

To add a little gap, give the image a style: style={{ marginRight: "0.5rem" }}

5. Duplicating for Email 📧

Copy the phone grid items and:

  • Change the source to emailIcon and alt="envelope".
  • Update the text to your email, e.g., zachary@gmail.com.

6. Aligning Text & Icons Properly 🎯

If text is too big, reduce by adding fontSize: "1rem" in the style of your Typography.
For the email icon, add verticalAlign: "bottom" so it lines up with the text baseline.

7. Save & Preview 💾

Save your file and refresh. You should see your icons and text nicely centered and aligned.
