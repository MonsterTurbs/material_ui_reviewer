1. What is a TextField? 🔤

The TextField lets users type text into forms. It wraps an input with labels, styling, and more built-in features! 😃

2. Variants 🎨

Three main looks:
• variant="standard"
• variant="filled"
• variant="outlined"
Choose the style that fits your design. 🖌️

3. Label, Value & Helper Text 🏷️

• label="Name" shows placeholder text that moves up as the field name.
• defaultValue="Hello" sets initial content inside.
• helperText="Some info" shows guidance below the field.

4. Required, Disabled & Field Types 🚫

• required adds a * and enforces input.
• disabled makes the field read-only.
• type="password" hides text.
• type="number" shows increment arrows only.🔒

5. Unique ID & Accessibility 🆔

Always give each TextField a unique id="myField" for screen readers and form labels. ✅

6. Error Prop & Validation 🚫

Set error to true to show a red outline and error styling. You control when to show it (e.g., if input is invalid). ⚠️

7. Multiline Fields 📝

• multiline turns it into a textarea.
• rows={4} fixes 4 lines tall.
• rowsMax={4} grows up to 4, then scrolls.📜

8. Select Prop (Dropdown) 🔽

Add select to create a dropdown menu with MenuItem options inside. Great for choosing from a list! 📝

9. Input Adornments & Icons 💡

Use InputAdornment via InputProps={{ startAdornment: ... }} or endAdornment to add icons or actions inside the field. 🔒

10. Full Width & Margin ↔️

• fullWidth makes the field fill its container.
• margin="normal" | "dense" adjusts spacing around the field.

11. Controlled vs Uncontrolled ⚙️

• Controlled: value comes from React state. Use for validation and data handling.
• Uncontrolled: Field manages its own value. Simpler but less flexible.

12. Underlying Components & Customization ⚙️

The TextField wraps FormControl, InputLabel, and Input. You can customize each via props or theme overrides for colors, fonts, and layouts. 🎨

13. Ready to Use! 🚀

Now you know all the key TextField features: labels, validation, multiline, adornments, and responsiveness. Let’s build your Contact & Estimate forms! 😊
