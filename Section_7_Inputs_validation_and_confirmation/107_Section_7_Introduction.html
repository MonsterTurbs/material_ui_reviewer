1. Start with a Row Container 🔲

This page has two side-by-side areas. Begin with:
<Grid container direction="row" spacing={4}>
This lays out left (form) and right (info) sections horizontally. 😊

2. Wrap Left Section for Alignment 🔧

To center the form vertically, wrap it in an extra Grid item before the container:
<Grid item>
<Grid container direction="column" alignItems="center">...form here...</Grid>
</Grid>
This lets you use alignItems on the wrapper. 👍

3. Add Page Title & Intro 📞

Inside the column container, add:
<Grid item><Typography variant="h2">Contact Us</Typography></Grid>
<Grid item><Typography variant="body1" paragraph>Have questions? We’re here to help!</Typography></Grid>
This shows the main heading and intro text. 📝

4. Display Phone & Email ⎆

For each contact method, use a small row:
<Grid item container alignItems="center" spacing={1}>
<Grid item><img src={phoneIcon} alt="phone" /></Grid>
<Grid item><Typography variant="body1">(123) 456-7890</Typography></Grid>
</Grid>
Repeat for email with emailIcon. 📧

5. Build the Form Fields 🖊️

Use Material-UI’s TextField inside a column container:
<Grid item container direction="column" spacing={2}>
<Grid item><TextField label="Name" fullWidth required /></Grid>
<Grid item><TextField label="Email" type="email" fullWidth required /></Grid>
<Grid item><TextField label="Phone" type="tel" fullWidth /></Grid>
</Grid>
These fields handle validation and styling for you. 🎉

6. Message Field & Send Button ✉️

Below the fields, add:
<Grid item><TextField label="Message" multiline rows={4} fullWidth /></Grid>
<Grid item><Button variant="contained" color="primary">Send Message</Button></Grid>
Now users can type and submit their message. 🚀

7. Create Sidebar Info Panel 📋

On the right, use another Grid item container to show a special call-to-action:
<Grid item container direction="column" spacing={2}>...text & button...</Grid>

This sits next to the form and invites action. 🤝

8. Add Padding for Clean Look 🧼

Give the form area extra white space:
style={{ padding: '4rem' }}
This ensures it doesn’t crowd the header or footer. ✨

9. Prepare Confirmation Dialog 💬

After clicking “Send Message,” show a Dialog (modal):
<Dialog open={open} onClose={handleClose}>...form recap & confirm...</Dialog>
This asks users to verify their info before sending. 📦

10. Save & Test ✅

Save your code, refresh, and try resizing or submitting. Ensure the form looks good, is easy to use, and that the confirmation dialog works smoothly. Great job! 🎉
