1. Add Space Above Buttons ✨

On the Grid container wrapping the two buttons, add: style={{ marginTop: '2rem' }} → now there's breathing room below the inputs!

2. Center Buttons Vertically 🎯

That same Grid container (default row) gets: alignItems="center" → vertically aligns Cancel & Send buttons in the middle.

3. Make Cancel Text Lighter 🎨

On the Cancel <Button>, add: style={{ fontWeight: 300 }} → gives a thin, minimal look.

4. Center Dialog Title 📝

On the <Typography variant="h4"> for “Confirm Message” add: align={matchesXS ? 'center' : undefined} → title sits perfectly centered!

6. Responsive Side Padding 🌐

For paddingLeft & paddingRight, use nested ternaries:

matchesXS ? '0' :
  matchesSM ? '5rem' :
    matchesMD ? '10rem' : '20rem'

Now the dialog adapts from tiny phones to large screens! 📱💻

7. See It in Action 🎉

  • ❇️ Large screens: plenty of white space all around.
  • 🔷 Medium: side margins shrink to 10rem.
  • 🔶 Small: side margins shrink to 5rem.
