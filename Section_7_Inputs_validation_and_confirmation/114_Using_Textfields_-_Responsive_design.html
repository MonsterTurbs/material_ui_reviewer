1. Stack Inputs & Full Width 🏗️

On the Grid container wrapping your inputs, add direction="column". Then on each TextField, add fullWidth so they fill the wrapper. Now they stack and center correctly.

2. Detect Medium Breakpoint 📏

Above your component, create:
const matchesMD = useMediaQuery(theme.breakpoints.down('md')); This lets you switch layout when the screen is medium or smaller.

3. Switch to Column on MD ↔️➡️⬇️

<Grid container
  className={classes.background}
  direction={matchesMD ? 'column' : 'row'}
>…</Grid>

All background content stacks on medium screens! 📱

4. Center Text at MD 🔠

For your headings/subtitles, add:
align={matchesMD ? 'center' : undefined} This centers the text only when stacked.

5. Mobile Background Image 📲

In your classes.background style, use a breakpoint override: 

[theme.breakpoints.down('md')]: {
  backgroundImage: \`url(\${mobileBackground})\`
}

6. Vertical Centering on MD ⬆️⬇️

When stacked, add:
justify={matchesMD ? 'center' : undefined} to push content to the vertical middle.

7. Add Vertical Margins on MD ↕️

  • On your Contact wrapper Grid: style={{ margin: matchesMD ? '5rem 0' : '0' }}
  • On each TextField Grid item: style={{ marginBottom: '0.5rem' }}

Gives breathing room when stacked! 🌬️

8. Input Font & Color Override 🎨

In theme.js under overrides, add: 

MuiInput: {
  root: {
    color: '#444',
    fontWeight: 300
  },
  underline: { … } // your existing underline overrides
}

Your typed text now matches your theme. 🖋️

9. Responsive Top Margin 🏞️

Tweak marginTop on your outer Grid with a nested ternary:
matchesSM ? '1rem' : matchesMD ? '5rem' : 0
This ensures consistent spacing on small & medium screens.
