1. Show an Error State 🚨

Add the error prop to a TextField:
<TextField error />
This turns the outline red to show something’s wrong! ❌

2. Display Helper Text ℹ️

Use helperText to explain the error:
<TextField error helperText="Please enter a name" />
Shows friendly instructions under the field. 😊

3. Track Error Messages in State 🧠

Create two pieces of state to hold any error messages:
const [emailHelper, setEmailHelper] = useState('');
const [phoneHelper, setPhoneHelper] = useState('');
Empty string = no error, any text = show error! ✍️

4. One OnChange Handler 🔄

Make a single function to check input as it changes: 

const onChange = (e) => {
  const { id, value } = e.target;
  let valid;
  switch(id) {
    case 'email':
      setEmail(value);
      valid = /…your email regex…/.test(value);
      setEmailHelper(valid ? '' : 'Invalid email');
      break;
    case 'phone':
      setPhone(value);
      valid = /…your phone regex…/.test(value);
      setPhoneHelper(valid ? '' : 'Invalid phone');
      break;
    default:
      break;
  }
};

5. Attach Handler & Props 🎯

  • Add onChange={onChange} to both Email & Phone fields.
  • Set error={emailHelper.length > 0} and helperText={emailHelper} on Email.
  • Do the same with phoneHelper on Phone.

6. Watch It Work! 🎉

- Type a wrong email → field turns red and shows “Invalid email.”
- Fix it → red outline and helper text disappear!
- Same for phone numbers. 👍
