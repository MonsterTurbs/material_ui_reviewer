⚙️ Responsive Fixes – Custom Software Page

Easy steps with emojis, ready for PDF! 📄✨

1️⃣ Add Bottom Space Below Animations 📏

  • On the container wrapping “Automation” + “UX” icons, add: style={{ marginBottom: '10rem' }}
  • Gives breathing room before the footer.

2️⃣ Hide Arrows on Mobile 🤫

  • Wrap each arrow’s <Grid item> in: <Hidden mdDown>…</Hidden>
  • They only show on desktop (md and up).

3️⃣ Center Header Text at md ⇩ 🖥️➡️📱

  • Detect medium-down with: const matchesMd = useMediaQuery(theme.breakpoints.down('md'))
  • On the top-row <Grid container>: 
    justify={matchesMd ? 'center' : undefined}
  • And each title/paragraph: 
    align={matchesMd ? 'center' : undefined}

4️⃣ Stack Doc + Scale Sections at md ⇩ ⬆️

  • On their row container: 
    direction={matchesMd ? 'column' : 'row'}
  • Align horizontally in column mode: 
    alignItems={matchesMd ? 'center' : undefined}
  • Add bottom spacing only when stacked: 
    style={{ marginBottom: matchesMd ? '15rem' : 0 }}

5️⃣ Add Extra Gaps at xs ↕️📱

  • Detect small-down: const matchesSm = useMediaQuery(theme.breakpoints.down('sm'))
  • On each icon container (in column): 
    
style={{
  marginTop: matchesSm ? '10rem' : 0,
  marginBottom: matchesSm ? '10rem' : 0
}}
  • Prevents icons from sticking together on tiny screens.
🎉 All set! Test at different widths before export. 👍
