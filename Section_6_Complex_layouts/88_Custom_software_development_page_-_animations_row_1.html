🖥️ Custom Software Dev Notes – Animations Row

Easy steps with emojis! 🌟

1️⃣ Update “Scale” Animation 🎨

  • Original pie-chart Lottie was flickering – replaced with a new smooth animation.
  • Design tweak: use a fresh “scale” animation that loops cleanly.

2️⃣ Add New Grid Row for Animations ➕

  • Under the existing icons row, insert: 
    <Grid item container direction="row">…</Grid>
  • This wraps two blocks: “Digital Documents & Data” and “Scale”.

3️⃣ Build “Digital Documents & Data” Block 📄

  • <Grid item container direction="column"> to stack title + paragraphs.
  • Use three `<Typography variant="body1" paragraph>…</Typography>` for text.
  • Import text from the design file to keep content accurate.
  • Limit width: add className={classes.itemContainer} with maxWidth: '40em'.

4️⃣ Import & Configure Lottie Animations 📦

  • Copy your defaultOptions block from LandingPage to CustomSoftware.js.
  • Import JSON data: 
    import documentsAnimation from '../animations/documentsAnimation/data.js';
  • Use them: 
    const documentsOptions = { animationData: documentsAnimation, loop: true, autoplay: true, rendererSettings: { preserveAspectRatio: 'xMidYMid slice' } };

5️⃣ Insert Lottie into Grid 🖼️

  • Next to text block, add: 
    <Grid item md>
  <Lottie options={documentsOptions} isStopped={true} />
</Grid>
  • Use md prop on each `Grid item` so they sit side-by-side until medium screen.

6️⃣ Mirror for “Scale” Section 🔄

  • Copy the entire “Documents” item and paste below.
  • Swap order: place the `Grid item` with Lottie first, then the text container.
  • Import your new scale JSON: 
    import scaleAnimation from '../animations/scaleAnimation/data.js';
  • Create const scaleOptions = { …, animationData: scaleAnimation } and use in Lottie.

7️⃣ Style Lottie Dimensions ⚙️

  • Adjust so your animations don’t get cut off:
    • <Lottie
  options={…}
  style={{
    maxHeight: 325,
    maxWidth: 275
  }}
/>
  • Experiment with values until the full animation shows. 🔍

8️⃣ Align “Scale” Text to Right ↪️

  • For headings/parags in the second block, add: 
    <Typography align="right" …/>
  • Keeps your text flush to the right of the row. 📐
🎉 You’ve set up the animation row! Next: animate interactivity & finish remaining blocks. 🚀
