🔧 Services Page: Spacing & Alignment Fixes

Easy steps—even a kid can follow! 🧒👶

1️⃣ Tweak Top Margin

Originally we had margin-top: 12M on .serviceContainer. Change it to 10M for a tighter top space:

.serviceContainer { margin-top: 10M; }

2️⃣ Section-Specific Top Spacing

The first block still sat too far down. Override just that one:

<Grid container
  className={classes.serviceContainer}
  style={{ marginTop: '5M' }}
> ...

3️⃣ Add Bottom Padding for Last Block

The Website section was pressed up against the footer. Add bottom margin:

<Grid container
  className={classes.serviceContainer}
  style={{ marginBottom: '10M' }}
> ...

4️⃣ Responsive Top Margin on Mobile 🤏

On small screens, shrink that 5M down to 1M:

style={{
  marginTop: matchesSM ? '1M' : '5M'
}}

5️⃣ Equalize Text-Block Widths for Alignment 📐

Give each text container a fixed width (35M) so both line up, but only on desktop:

style={{
  textAlign: 'left',
  width: matchesSM ? undefined : '35M'
}}

Copy into both the iOS block and Website block’s text‐wrapper.

6️⃣ Match Icon Widths 🔗

Because the icons differ in natural size, force both to 250M:

<img
  src={mobileIcon}
  style={{ width: '250M' }}
/>

Do the same for the website icon.

7️⃣ Add “Services” Heading

  • Insert a `` with `Services` above the blocks
  • Give it left margin (5M) on desktop, none on mobile
  • Center text on mobile with ``
  • Adjust top margin (e.g. 2M desktop → 1M mobile) via media query
🚀 All set! Now your Services page has perfect, consistent spacing and alignment. 🎉
