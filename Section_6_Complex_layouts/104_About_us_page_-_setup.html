1. Create About.js 📁

In your components folder, add a new file named About.js. This will hold the About Us page code. 😊

2. Import Tools 📥

At the top of About.js, import:
import React from 'react';
import { makeStyles, useTheme } from '@material-ui/core/styles';
import Grid from '@material-ui/core/Grid';
import Typography from '@material-ui/core/Typography';
These give you styling hooks, grid layout, and text components. 🛠️

3. Setup Styles & Component 🎨

Define styles and component:
const useStyles = makeStyles(theme => ({ ... }));
Export a function About, inside get classes = useStyles() and theme = useTheme(). Then return <Grid container direction="column"> wrapping the page. 💅

4. Add Title 🏷️

First child of container:
<Grid item className={classes.rowContainer} style={{ marginTop: '2m' }}>
<Typography variant="h2">About Us</Typography>
</Grid>
This displays the page title with side padding and top margin. 🎉

5. Mission Statement ✒️

Create a centered mission text:
<Grid item container justify="center" className={classes.rowContainer}>
<Typography variant="h4" align="center" className={classes.missionStatement}>"Your mission text..."</Typography>
</Grid>
Style missionStatement with fontStyle: 'italic', fontWeight: 300, fontSize: '1.5rem', maxWidth: '50m', lineHeight: 1.4. 😊

6. Register in App.js 🔗

In App.js, replace the dummy route:
import About from './components/About';
<Route path="/about" render={() => <About />} />
Now your About page loads! 🚀

7. History Section 📜

Below mission, add:
<Grid item container direction="row" className={classes.rowContainer}>
<Grid item container direction="column" lg>
<Typography variant="h4" gutterBottom>History</Typography>
<Typography variant="body1" paragraph style={{ fontWeight:700, fontStyle:'italic' }}>"We’re the new kid on the block..."</Typography>
<Typography variant="body1" paragraph>Other history sentence...</Typography>
...
</Grid>
<Grid item lg container justify="center">
<img src={historyIcon} alt="quill on book" style={{ maxHeight:'22m' }} />
</Grid>
</Grid>
• Import historyIcon from your assets.
• Add lg props on both items to share space, then center icon with justify="center". 📖

8. Constrain Text Width ✂️

On the Grid item container direction="column" for text, add style={{ maxWidth: '35m' }} so paragraphs stay readable. 📐

9. Space-Around Alignment ⚖️

Wrap each of the two containers (text & icon) in an extra <Grid item>. Then on their row wrapper, add justify="space-around" to evenly distribute them. 🎯

10. Save & Review ✅

Save your code, refresh the browser, and test at various sizes to ensure history section looks balanced and responsive. Great job! 🎉
