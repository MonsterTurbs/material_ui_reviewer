1. Make the File 📁

Create a new file Revolution.js inside your components folder. This will hold your Revolution page code. 😊

2. Import Stuff 📥

At the top of Revolution.js, import React, makeStyles, useTheme, Grid, and Typography from Material-UI. This gives you the tools to build and style the page. 🛠️

3. Setup Styles & Component 🎨

Use makeStyles(theme => ({ ... })) to define your useStyles hook. Export a component Revolution, get classes & theme, and return a <Grid container direction="column">. 💅

4. Add Page Title 🏷️

Inside the grid, add:
<Grid item><Typography variant="h2">The Revolution</Typography></Grid>
Then in App.js, import and render Revolution instead of the dummy. 🚀

5. Style the Title ✍️

Give the title a special font and spacing:
style={{ fontFamily: 'Pacifica', marginTop: '2m' }} className={classes.rowContainer}. This matches the design. 🎉

6. Vision Section 🔭

Add a row for Vision:
<Grid item container direction="row" className={classes.rowContainer}>
• Left: <Grid item lg><img src={vision} alt="mountain through binoculars"/></Grid>
• Right: <Grid item container direction="column" lg> with a H4 subtitle and five body1 paragraphs. 📄

7. Share & Center Space ⚖️

Add lg to both grid items so they split space equally on large screens. Use alignItems="center" on the row to center the icon vertically. Give text & icon maxWidth: '40m' and icon a marginRight: '5m'. 👍

8. Swap for Technology 💻

Copy the Vision row and paste below. Swap text to left, animation to right. Change title to “Technology” and paste its paragraphs. Remove any align="right". 🎉

9. Add Lottie Animation 🎥

Import Lottie and defaultOptions. Replace the <img> with:
<Lottie options={defaultOptions} isStopped style={{ maxWidth: '40m', margin: 0 }} />
Then set row justify="flex-end". 🌟

10. Final Tweaks & Test 📱

Save and refresh! Check large & small screens. Items share space on big, stack on mobile. Responsive basics done! 🎉

11. Add Process Title 📝

Under the last grid item, add:
<Grid item container direction="row" className={classes.rowContainer} justify="center">
  <Grid item><Typography variant="h4" gutterBottom>Process</Typography></Grid>
</Grid>
This centers the Process heading. 🎯

12. Create Consultation Block 🤝

Below the title, add another Grid item container direction="row" className={classes.rowContainer}. Inside:

• A Grid item container direction="column" lg> for text, with:
- <Typography variant="h4" gutterBottom>Consultation</Typography>
- Three <Typography variant="body1" paragraph> paragraphs (copy from design file). 📝

13. Style Colors & Background 🎨

Make title black: style={{ color: '#000000' }}. Paragraphs white: style={{ color: '#FFFFFF' }}. On the row container, add:
style={{ backgroundColor: '#B3B3B3', height: '90m' }} for the gray block. 🚧

14. Adjust Spacing & Width 📐

Add style={{ marginTop: '5m' }} to the title and style={{ maxWidth: '20m' }} to paragraphs so text has room to breathe. ✂️

15. Add Icon 🖼️

Next to the text column, add:
<Grid item lg><img src={consultation} alt="handshake"/></Grid>
Import the icon:
import consultation from './assets/consultation-icon.svg';. 🤝

16. Share Space & Align Icon ⚖️

Add lg props so text & icon share space. Use style={{ alignSelf: 'center' }} on the icon's grid item to center it vertically without moving the text. Perfect! 👍

17. Next: Responsive Design 📱

All process steps built! In the next video, use useMediaQuery and theme breakpoints to stack blocks on mobile and keep it looking great everywhere. 🚀

18. Import Remaining Step Icons 📥

Under your consultation import, add:
import mockup from './assets/mockup-icon.svg';
import review from './assets/review-icon.svg';
import design from './assets/design-icon.svg';
import build from './assets/build-icon.svg';
import launch from './assets/launch-icon.svg';
import maintain from './assets/maintain-icon.svg';
import iterate from './assets/iterate-icon.svg';
These icons represent each process step. 🎯

19. Duplicate Consultation Block 🔁

Copy the entire Grid item container used for Consultation and paste it below. You'll create one block each for Mockup, Review, Design, Build, Launch, Maintain, and Iterate. 🔄

20. Customize Each Block ✏️

In each new block:
- Change the title (e.g., Mockup)
- Swap src to the correct icon and update alt text (e.g., "basic website design outline")
- Copy in the matching paragraphs from the design file
This sets up all step descriptions. 📝

21. Assign Background Colors 🎨

On each rowContainer, adjust the style:
- Mockup: backgroundColor: '#FF7373'
- Review: backgroundColor: '#39B54A'
- Design: backgroundColor: '#A67C52'
- Build: backgroundColor: '#FBB03B'
- Launch: backgroundColor: '#C1272D'
- Maintain: backgroundColor: '#8E45C2'
- Iterate: backgroundColor: '#29ABE2'
Give each section its unique color and set a fixed height if needed. 🎨

22. Save & Verify ✅

Save your code and refresh the page. Scroll through each process step to confirm titles, icons, text, and background colors all look correct. You’ve completed the Process section! 🎉
