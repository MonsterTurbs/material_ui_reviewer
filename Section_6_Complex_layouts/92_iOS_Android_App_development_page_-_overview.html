📱 iOS & Android App Dev Page

Simple layout breakdown! 🌟

1️⃣ Page Wrapper

  • One big container for everything:
  • <Grid container direction="column">
  • Stacks each section top-to-bottom. 📚

2️⃣ Top Navigation Row ↔️

  • Uses a row container:
  • <Grid item container direction="row">
  • Contains three items:
    • ◀️ Back arrow
    • 📄 Text (title + description)
    • ▶️ Forward arrow
  • Text itself is a column: <Grid container direction="column">
  • So lines of text stack nicely. 🆙⬇️

3️⃣ Main Content Sections

  • Below header, three side-by-side blocks:
  • Each block is: <Grid item container direction="column">
  • Inside each:
    • 🔸 A title: <Typography variant="h4">
    • 🔸 A paragraph: <Typography variant="body1">
  • The middle block swaps order:
    • Text → Animation → Text

4️⃣ Feature Icons Row 🎨

  • A row of 4 icon cards:
  • <Grid container direction="row" justify="space-around">
  • Each icon card is: <Grid item container direction="column">
  • Inside:
    • 🔹 Title (h4)
    • 🔹 Image icon

5️⃣ Mobile Responsiveness 📱

  • Hide arrows on phones: <Hidden mdDown>…</Hidden>
  • Center text on small: align={matchesSm?'center':'left'}
  • Stack side-by-side blocks into columns: direction={matchesMd?'column':'row'}
  • Simple breakpoints keep it looking great!
