📱 Page Overview

We’re building the “iOS & Android App Development” page. It’s like the other service pages: a header with arrows to hop between services, and below it three sections, each with an icon and some text. Super simple! 😊

🔝 Header Section

  • Use a Grid container with direction="column".
  • First row has three items:
    1. 🔙 Back arrow (go to previous service)
    2. 📄 Text block (title + paragraphs)
    3. 🔜 Forward arrow (go to next service)
  • Use Material UI’s IconButton wrapping an img for each arrow.
  • Typography:
    • variant="h2" for the big title
    • variant="body1" for the description text

🔥 Main Content Row

  • Wrap in a Grid item container with direction="row".
  • Three parts side by side:
    1. Left: text block (title & paragraphs)
    2. Center: Lottie animation
    3. Right: text block (title & paragraphs)
  • Make them share the space until medium screens (md breakpoint), then stack on smaller.
  • Use responsive props:
    • md to share space
    • direction={matchesMd ? "column" : "row"} to switch layout

✅ Icon + Text Blocks

  • Another row: Grid container direction="row".
  • Three icon blocks, each:
    1. Grid item container direction="column"
    2. First item: Typography variant="h4" for the small title
    3. Second item: <img> icon
  • Center content with alignItems="center" and justifyContent="center".
  • Make them share space until md, then stack and add spacing (marginTop). 📐

👉 Call to Action

At the very bottom, add the CallToAction component for the free estimate button. It’s full-width and links to “/estimate”. 🎉
