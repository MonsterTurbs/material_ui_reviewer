Add Space Between Sections 🌟

Give each section breathing room by adding style={{ marginTop: '5m' }} on the Vision row container, and style={{ marginTop: '10m', marginBottom: '10m' }} on the Technology row container. Nice gaps! 😊

Import Breakpoints 🔍

At the top of Revolution.js, import:
useMediaQuery from Material‑UI:
import useMediaQuery from '@material-ui/core/useMediaQuery';
Then inside the component, add:
const matchesMD = useMediaQuery(theme.breakpoints.down('md'));
const matchesSM = useMediaQuery(theme.breakpoints.down('sm'));
These tell you if the screen is medium or small. 📱

Responsive Icon Size 📐

For each <img>, change maxWidth using a ternary:
style={{ maxWidth: matchesSM ? '300px' : '40m' }}
This keeps icons big on desktop and smaller on phones. 👍

Center Title on Smaller Screens 🎯

Add align={matchesMD ? 'center' : undefined} on your <Typography variant="h2"> so the Revolution title jumps to center when the screen narrows. ✨

Switch Row to Column 🛠️

On each section’s row container, replace direction="row" with:
direction={matchesMD ? 'column' : 'row'}
This stacks text and icons on medium & smaller screens. 🔄

Reset Icon Margin 🔄

Icons had marginRight: '5m' for desktop. Change to:
style={{ marginRight: matchesMD ? 0 : '5m', marginBottom: matchesMD ? '5m' : 0 }}
This zeroes out desktop margin but adds bottom space when stacked. 🎉

Center Text on Medium & Below 📝

For each Typography, use:
align={matchesMD ? 'center' : undefined}
This makes all headings and paragraphs centered on smaller screens. 😊

Center Lottie on Small Screens 🎬

Replace justify="flex-end" with:
justify={matchesMD ? 'center' : 'flex-end'}
This keeps your animation centered when stacked. Perfect! 💫

Align Process Text Horizontally 📏

On each Grid item container direction="column" in the Process steps, add:
alignItems={matchesMD ? 'center' : undefined}
Centers text blocks inside their column containers. 👍

Stack Process Blocks Responsively 📱

Switch each Process block’s row container to:
direction={matchesMD ? 'column' : 'row'}
This ensures icons and text never fight for space! 😊

Make Icons Fully Responsive 🌈

To each <img>, add:
style={{ width: '100%', maxWidth: '700px' }}
Icons now fill their container and cap at a nice maximum width. No scroll bars! 🚀
