📝 Services Page – Animation & Sections

Simple language + emojis, ready for PDF! 🎉

1️⃣ Fix Lottie Dimensions 📏

  • Use style={{ maxHeight: 275, maxWidth: 275, minHeight: 250 }} so animations aren’t clipped.
  • Adjust numbers by trial until full animation shows. 🔍

2️⃣ Evenly Space Two Animations 🤹‍♂️

  • On the row container, set justify="space-around" to spread them horizontally.
  • Apply md prop on each animation’s <Grid item> so they stay side-by-side until the medium breakpoint.

3️⃣ Break to Column at Medium 🔄

  • Still side-by-side until medium; then switch to column to stack:
    • direction={matchesMd ? 'column' : 'row'}
  • Avoid overlap at awkward widths. 👍

4️⃣ Add “Root Cause Analysis” Section 🌳

  • New row: <Grid item container direction="row">
  • Left item: image of roots (set height/width), then right item:
    • <Typography align="center" variant="h4">…</Typography>
    • Three body1 paragraph tags with align="center".
  • Center with alignItems="center" on the container.

5️⃣ Copy & Create “Automation” + “UX Design” Rows 🤖🎨

  • Duplicate the Digital-Docs/Scale grid item container.
  • For each:
    • Change title & paragraphs from design file.
    • Import new JSON: 
      import automationAnimation from '../animations/automationAnimation/data.js';
      import uxAnimation from '../animations/uxAnimation/data.js';
    • Copy Lottie options block, rename to automationOptions & uxOptions.
    • Paste animations into grid items, adjust style dims (maxHeight, maxWidth).

6️⃣ Margin Buffers for Middle Sections 🌟

  • Add spacing above/below sections with: 
    style={{ marginTop: '20rem', marginBottom: '20rem' }}
  • Makes page feel open and avoids clutter. 😊
🚀 You’re all set! Review responsiveness and finalize before PDF export. 🎨
