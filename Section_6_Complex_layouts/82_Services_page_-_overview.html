🛠️ Services Page Overview

Simple breakdown for kids or beginners, with fun emojis! 🚀

📄 1. Overall Wrapper

The whole page lives inside a Grid container set to direction="column". That stacks each big section (title, iOS block, Android block, etc.) top-to-bottom.

🔢 2. Main Sections

Inside our column container we have 4 main Grid items:

  • 🌟 Title (“Our Services”)
  • 📱 iOS & Android Apps
  • 💻 Custom Software
  • 🌐 Website Development

➡️ 3. Service Block Layout

Each individual service block is a Grid container (default row):

  • Grid item: text on the left (title, subtitle, “Learn More” button) 📝
  • Grid item: icon on the right (e.g. phone icon) 📷

🔠 4. Inside the Text Area

That left-side text area is itself a Grid container with direction="column":

  • 1️⃣ Grid item: title (Typography variant="h4")
  • 2️⃣ Grid item: subtitle (e.g. “Save time, save money”)
  • 3️⃣ Grid item: second subtitle
  • 4️⃣ Grid item: “Learn More” button

⚙️ 5. Reuse & Adjust

Because it’s the same pattern on Home, just copy your Home’s service block component and:

  • 🖊️ Change icon & texts
  • ↔️ Adjust justify to left/right as needed
  • 📱 Keep breakpoints for mobile stacking

✅ 6. Ready to Code!

With this map in hand, jump into your code editor and build each <Grid container> & <Grid item> one by one. You’ve got this! 🌟

Happy coding! 🎉
