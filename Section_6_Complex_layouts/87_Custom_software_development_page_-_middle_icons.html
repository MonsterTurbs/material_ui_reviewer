📒 Custom Software Dev Notes – Part 2

Simple steps even a beginner can follow! 🌟

1️⃣ Limit Paragraph Width

  • Add className={classes.heading} on your text container. 📏
  • In useStyles, set heading: { maxWidth: '40em' } so lines don’t stretch too far. ✂️

2️⃣ Center the Arrows

  • Wrap each arrow’s <Grid item> in className={classes.arrowContainer}. 🎯
  • Style it with marginTop: '0.5em' to vertically align with the title. ↕️
  • Remove the hover background on the arrow buttons by adding backgroundColor: 'transparent'. 🚫🎨

3️⃣ Fix Text Variant in Theme

  • Oops! If body1 isn’t defined, add it back in theme.js:
    • body1: { fontSize: '1.25rem', color: '#444', fontWeight: 300 }
  • Ensures your paragraphs look correct. ✅

4️⃣ Add Page Padding & Border Alignment

  • On the main grid container, add className={classes.mainContainer}. 🏠
  • Give it padding: {`padding: '2em 5em 10em'`} to keep content off the edges. 📐
  • Shift the left arrow into the padding by styling its container: marginRight: '1em', marginLeft: '-3.5em'. ↩️

5️⃣ Wire Up Arrow Navigation

  • Use React-Router’s component={Link} & to="/…" on each <IconButton>. 🔗
  • Also call props.setSelectedIndex(…) in onClick to keep your menu highlighted. 🎛️

6️⃣ Layout the Three Icons Row

  • Wrap icons in a Grid item container direction="row". 🧩
  • Each icon block: <Grid item container direction="column" md>…</Grid> • md makes them share width until medium breakpoint. 📱➡️🖥️
  • Limit their width with style={{maxWidth: '40em'}} so they don’t stretch too far. 📏
  • Center them with justify="center" & alignItems="center". 🎯
  • Add marginTop: '1.5em', marginBottom: '2em' around the row for breathing room. 🌬️
🚀 You’ve shaped up the Hero & Icons! Next: finish animations & process blocks. 👍
