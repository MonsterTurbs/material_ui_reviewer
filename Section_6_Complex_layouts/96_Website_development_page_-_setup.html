1. Create File 📁

Make a new file named websites.js inside the components folder. This will hold our website page code. 😊

2. Import Modules 📥

At the top of websites.js, import React, Material‑UI tools (Grid, IconButton, Typography, Hidden), makeStyles, useTheme, and useMediaQuery. Also import back and forward arrow icons. These let us build and style our page. 🛠️

3. Setup Styles & Component 🎨

Use makeStyles to create a useStyles hook for styling. Then define the Websites function component, get classes and theme, and return a Grid container with direction="column". 💅

4. Copy Header & Arrows 🔁

Copy the header & arrow Grid item from the Custom Software page. Paste it into websites.js. This makes the page title and navigation arrows appear. ↔️

5. Add Component to App.js ➕

Open App.js, import Websites from './websites', and add its route where the other service pages are rendered. Now the website page can load! 🌐

6. Style Header 🖌️

Copy the heading styles from the Custom Software page into your useStyles object and save. Refresh to see the styled title on the Website Development page. ✨

7. Update Text 📝

Open the design file, copy the two paragraph texts under Website Development, and replace the old text in websites.js. Remove any extra Typography components. This puts the right content on the page. 📄

8. Add Content Row ➗

Under the previous Grid item, add a new Grid container direction="row" with two Grid items. This sets up a row with an icon on the left and text on the right. 📐

9. Import Images & Analytics 🖼️

Import images: analytics, SEO, outreach, and ecommerce from your assets folder. These icons will go in each section. 📂

10. Add Analytics Section 📊

Inside the first Grid item, add a Typography variant="h4" with text "Analytics" and a <img> showing the analytics icon. This displays the section title and image. 🔍

11. Align & Style Text & Items 🎯

On the row container, set alignItems="center". For the paragraph, add a class with maxWidth: '30m' to limit its width. Save and refresh to see centered icon and text nicely side by side. ⚖️

12. Adjust Image Position 📐

Add a style of marginLeft: '-2.75m' to the analytics image to align the graph bar with the title and let the magnifying glass hang over. Looks neat! 📏

13. Repeat for Other Sections 🔄

Copy the analytics section code, change the icon, title, and text for SEO, Outreach, and eCommerce. Each section uses the same layout. You're done! 🎉
