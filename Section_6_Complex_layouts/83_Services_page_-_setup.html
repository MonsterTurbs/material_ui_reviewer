🛠️ Services Page Study Notes

Easy steps for beginners, full-width cards & fun emojis! 🌟

1️⃣ Create File & Setup

• In your components folder, make Services.js 📁
• Start React dev-server so you can preview as you code 🚀

2️⃣ Add Basic Imports

  • Import React, Link from React-Router
  • Bring in Material UI hooks & components:
    makeStyles, useTheme, useMediaQuery,
    Grid, Typography, Button
  • Create useStyles for your CSS‐in‐JS styling 🎨

3️⃣ Copy & Paste Blocks

• On your landing page you already built a “Services” block 🏗️
• Copy the three service sections (iOS/Android, Custom Software, Websites) – the whole
<Grid container>…</Grid> snippet – into Services.js

4️⃣ Hook into App.js

  • Import your new Services component in App.js 🔗
  • Replace the dummy `` for “/services” with a render={props => <Services {...props} />}
  • This makes the page actually show up at /services 🎉

5️⃣ Import Missing Assets & Hooks

  • Bring in your three icons:
    import customIcon from "../assets/customSoftware.svg";
    import mobileIcon from "../assets/mobileApps.svg";
    import webIcon from "../assets/websiteIcon.svg";
  • Import useTheme & useMediaQuery to handle responsive tweaks 📱
  • Import your ButtonArrow helper for the “Learn More” arrows ➡️

6️⃣ Copy & Apply Styles

• From your LandingPage styles, copy over:

  • serviceContainer (margins),
  • icon spacing,
  • subtitle text style,
  • specialText (orange highlight),
  • learnButton style 🎨

7️⃣ Mirror Layout & Margins ↔️

  • Swap block order: bring “Custom Software” into the middle
  • For the right-side “Website” block, add justify="flex-end" above small size
  • Remove its marginLeft and add marginRight so it matches spacing
  • Use media queries (matchesSmall) to keep it responsive 📐
🚀 You’re all set! Now test at different screen sizes and celebrate your responsive grid skills! 🎉
