1. Create Team Section Container 👥

Under the History section, add a new container:
<Grid item container direction="column" className={classes.rowContainer}>
This will stack all Team items vertically. 😊

2. Add 'Team' Title 🏷️

Inside the container, add:
<Grid item><Typography variant="h4" gutterBottom align="center">Team</Typography></Grid>
This centers the 'Team' heading with space below. ✨

3. Add Name & Founder Bio 📝

Add your introduction paragraphs:
<Grid item><Typography variant="body1" paragraph align="center">My name is Zachary Reese, Founder.</Typography></Grid>
<Grid item><Typography variant="body1" paragraph align="center">I started coding when I was nine years old.</Typography></Grid>
Both are centered. 📄

4. Import Avatar & Images 📥

At top of About.js, import:
import Avatar from '@material-ui/core/Avatar';
import profile from './assets/founder.jpeg';
import yearbook from './assets/yearbook.jpg';
import puppy from './assets/puppy.jpg';
Avatar creates a circular image container. 🎨

5. Display Profile Avatar 🐶

Add your picture:
<Grid item><Avatar alt="Founder" src={profile} className={classes.avatar} /></Grid>
In styles, add:
avatar: { width: '25m', height: '25m' } in useStyles. 😊

6. Center Team Content 🎯

On the Grid container direction="column", add:
alignItems="center"
This centers avatar and text horizontally. 👍

7. Create Fun Fact Row 📊

Under bio, add a row:
<Grid item container direction="row" className={classes.rowContainer}>
This holds Yearbook, Bio Text, and Puppy blocks side by side. 🔄

8. Yearbook Block 📖

Add Yearbook image & caption:
<Grid item container direction="column" lg>
<img src={yearbook} alt="yearbook page about founder" style={{ width: '100%', maxWidth: '700px' }} />
<Typography variant="caption">A page from my sophomore yearbook</Typography>
</Grid>

9. Bio Text Block 🗒️

Add your biography text:
<Grid item lg>
<Typography variant="body1" paragraph align="center" style={{ maxWidth: '45m', padding: '1.25m' }}>Your bio paragraph here.</Typography>
</Grid>

10. Puppy Block 🐕

Add puppy image & caption:
<Grid item container direction="column" lg alignItems="flex-end">
<img src={puppy} alt="grey spotted puppy" style={{ width: '100%', maxWidth: '700px' }} />
<Typography variant="caption">My miniature dapple dachshund Sterling</Typography>
</Grid>

11. Share Space with lg 🚀

Add lg to Yearbook, Bio, Puppy blocks so they split evenly on larger screens. 📐

12. Customize Caption Style 🎨

In theme.js, update caption variant:
caption: { fontSize: '1rem', fontWeight: 300, color: '#555' }
This styles all captions consistently. ✨

13. Save & Verify ✅

Save your changes, refresh the browser, and resize the window to ensure everything is centered, spaced well, and responsive. Well done! 🎉
