📝 Custom Software Dev Page Study Notes

Easy for kids & beginners! 🚀

1️⃣ File & Imports

  • Make a new file: CustomSoftware.js 📂
  • Import React, Grid, Typography, IconButton, Lottie, Link, makeStyles, useTheme, useMediaQuery 😊
  • Also import your back-arrow & forward-arrow icons from /assets 👈👉

2️⃣ Hook Up Styles

  • Create useStyles with makeStyles(theme => {{…}}) 🎨
  • Inside the component, call const classes = useStyles() ✅
  • Also get the theme & media query: const theme = useTheme(); const matchesSM = useMediaQuery(theme.breakpoints.down('sm')); 📏

3️⃣ Main Grid Container

  • Wrap the whole page in: <Grid container direction="column">…</Grid> 🏗️
  • This stacks sections vertically.

4️⃣ Top Arrow + Text Section

  • Add a row container for nav: <Grid item container direction="row">…</Grid> ↔️
  • First item: <Grid item><IconButton><img src={backArrow}/></IconButton></Grid> 🔙
  • Middle item (column): • Title: <Typography variant="h2">Custom Software Development</Typography> • 3 paragraphs: <Typography variant="body1">…</Typography> 📄
  • Last item: <Grid item><IconButton><img src={forwardArrow}/></IconButton></Grid> 🔜
