1. What Is Axios? 🤔

Axios is a JavaScript library to make HTTP requests easily in React. We’ll use it to call our Firebase function URL and send emails. 🚀

2. Install Axios 📥

In your project root (not the functions folder), run:

npm install axios

Now you can import it in your React code!

3. Import & Setup 🛠️

At the top of Contact.js, add:

import axios from 'axios';

4. Create an onConfirm Function ✍️

Above your return statement, define:

const onConfirm = () => {
// we’ll call Axios here
};

5. Get Your Function URL 🔗

  • Open Firebase Console → Functions → copy your sendMail URL.
  • This is where Axios will send the GET request.

6. Call Axios.get() 🎉

Inside onConfirm, write:

axios.get('YOUR_FUNCTION_URL_HERE')
.then(response => console.log(response))
.catch(error => console.log(error));

This triggers your cloud function and logs the result. 👍

7. Test It! 🧪

  • Fill in name, email, phone, message on Contact page.
  • Click Send Message to open dialog.
  • Click Confirm to run onConfirm.
  • Open DevTools → Network tab to see the request and 200 status.

A 200 response and “Message sent successfully” means it worked! 📬
