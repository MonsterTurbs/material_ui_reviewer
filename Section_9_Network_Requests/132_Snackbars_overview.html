1. What Is a Snackbar? 🔔

A Snackbar is a small message box that pops up to tell the user about something that happened, like “Message sent!” or “Error occurred.” It appears briefly and then disappears automatically. 😊

2. Basic Usage 📦

Import and render a Snackbar with just a message:

import React, { useState } from 'react';
import Snackbar from '@material-ui/core/Snackbar';

function MyComponent() {
const [open, setOpen] = useState(false);

return (
<>
<Button onClick={() => setOpen(true)}>Show Message</Button>
<Snackbar
open={open}
message="Hello from Snackbar!"
onClose={() => setOpen(false)}
/>
</>
);
}

3. Control Visibility & Auto Hide ⏱️

  • open: boolean state to show/hide.
  • onClose: callback when it should close.
  • autoHideDuration={3000}: milliseconds before it closes itself.

4. Add an Action Button 🎬

You can include buttons like “UNDO” or a close icon:

<Snackbar
open={open}
message="Deleted item"
action={(
<Button color="secondary" size="small" onClick={handleUndo}>
UNDO
</Button>
)}
onClose={() => setOpen(false)}
/>

5. Positioning the Snackbar 📍

Use anchorOrigin to choose where it appears:

<Snackbar
open={open}
message="Saved!"
anchorOrigin={{
vertical: 'bottom',
horizontal: 'right'
}}
onClose={() => setOpen(false)}
/>

Options: top/bottom and left/center/right. 👍

6. Change Transitions & Styles 🎨

  • TransitionComponent={Slide} or Fade for animations.
  • Customize colors via theme overrides or ContentProps.
