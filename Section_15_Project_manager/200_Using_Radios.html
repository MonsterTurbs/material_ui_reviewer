1. What are Radio Buttons? ü§îüîò

Radio buttons let you pick one choice out of many. When you click one, any other turns off. Perfect for questions like "Which service?"

2. Importing in React üõ†Ô∏è

First, bring in the components from Material UI at the top of your file:

import Radio from '@material-ui/core/Radio';
import RadioGroup from '@material-ui/core/RadioGroup';
import FormControlLabel from '@material-ui/core/FormControlLabel';

3. Setting Up State üìã

Use React useState to keep track of the selected value:

const [service, setService] = useState('');

Pass value and onChange to the group:

<RadioGroup
  value={service}
  onChange={e => setService(e.target.value)}
> ... </RadioGroup>

4. Adding Labels & Accessibility üè∑Ô∏è

Wrap each Radio in a FormControlLabel to add text labels and keep it accessible:

<FormControlLabel
  value="website"
  label="Website"
  control={<Radio />} 
/>

This shows the label next to the button and helps screen readers.

5. Styling & Alignment üé®

You can customize spacing and look by wrapping your radio set in a <Grid> or using CSS:

  • Use labelPlacement="start" to put labels on the left.
  • Wrap in <Grid container spacing={2}> for even gaps.
  • Override classes in your theme for colors or font weights.

Example override to thin labels:

overrides: {
  MuiFormControlLabel: {
    label: { fontWeight: 300 }
  }
}
