📅 What Are Material UI Pickers?

Pickers let users choose dates or times with a nice calendar/clock UI instead of typing! 🎉

📦 1. Install Packages

  • Run npm install @material-ui/pickers to get the picker components.
  • Run npm install date-fns for date handling utilities. 📆

🛡️ 2. Wrap with Provider

In your top-level (e.g. _app.js), wrap UI with:

import { MuiPickersUtilsProvider } from '@material-ui/pickers';
import DateFnsUtils from 'date-fns';

<MuiPickersUtilsProvider utils={DateFnsUtils}>
  <App />
</MuiPickersUtilsProvider>

📍 This makes pickers work anywhere inside.

🗓️ 3. Use the DatePicker

Import and add a date picker in your component:

import { DatePicker } from '@material-ui/pickers';

const [selectedDate, setSelectedDate] = useState(new Date());

<DatePicker
  label="Pick a date"
  value={selectedDate}
  onChange={date => setSelectedDate(date)}
  animateYearScrolling
/>

Now clicking opens a calendar! 🗓️

⏰ 4. Other Picker Types

  • TimePicker for selecting times ⏱️
  • DateTimePicker combines date & time 🕒

🎨 5. Theming & Styles

Pickers use your Material UI theme colors automatically. 🔵

🚀 6. Try It Out!

  • Restart your app (npm run dev).
  • Place a picker, pick a date or time, and watch state update! 🥳
