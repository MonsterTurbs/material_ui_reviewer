📁 What is Our Second Project?

We’re building a Project Manager tool to organize all the estimates from our website. It’s an internal app with tables, filters, and dialogs. 🗂️

🧩 Key Features

  • 🔍 Search Box: Type to filter projects by name, service, or platform.
  • 🔀 Switch Filters: Toggle services (Websites, Software, Mobile) on/off.
  • 💵 Price Filter: Choose >, <, or = and show a chip with the selected range.
  • ➕ Add Project: Click the plus icon to open a dialog and enter new project details.

📊 Table & Controls

  • ✅ Checkboxes: Select projects to delete, with an undo snack bar.
  • ↕️ Sortable Columns: Click headers to sort A→Z or Z→A.
  • 📑 Material UI Table: Shows project name, service, features, date, users, and total cost.

🗣️ Add Project Dialog

  • 🗓️ Date Picker: Choose the project date using Material UI Pickers.
  • ✏️ Text Fields: Enter project name and total cost.
  • 🔘 Radio Groups: Select service, complexity, and user count.
  • 📋 Select Dropdown: Pick features or platforms.
  • ✅ Cancel & Add buttons to close or save.

🔧 Layout Tips

  • 📐 Use a Grid container direction="column" for main layout.
  • 📦 Wrap each section (search, switches, filters, table) in Grid item.
  • ↔️ For dialog, use a row container with three column items for form sections.
  • 🎨 Input Adornments let you add icons inside TextFields (plus, filter symbols).
