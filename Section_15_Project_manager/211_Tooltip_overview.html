💡 What is a Tooltip?

A tooltip is a small box that shows helpful text when you hover over, focus on, or tap an element.

👆 How to Add a Tooltip

Wrap any component (like an icon button) inside the <Tooltip> component.

Use the title prop to set the text you want to show.

🎯 Placement Options

You can choose from 12 positions (top, bottom, left, right, and their variations) to place the tooltip exactly where you need it.

🎨 Custom Styles

Use CSS (e.g., background color, opacity, borders) to style your tooltip via the classes prop.

You still pass your styles through the title prop!

➡️ Arrow Prop

Add the arrow prop to show a small arrow pointing to the element.

⚙️ Trigger Events

Tooltips can open on hover, focus, touch, or click.

You can disable specific events using props like disableHoverListener.

🔄 Controlled Tooltips

You can control when a tooltip opens by using the open, onOpen, and onClose props in your component logic.

📐 Width Customization

The default max width is 300px. Change or disable it via the classes.tooltip prop to fit longer text.

🖱️ Interactive Tooltips

Use the interactive prop to keep the tooltip open when you hover over it, allowing extra content inside.

🚫 Disabled Elements

Wrap disabled buttons in a <span> to enable tooltips on them (disabled elements don’t fire events).

✨ Transitions & Delays

Choose between Grow, Fade, or Zoom transitions via the TransitionComponent prop.

Set enterDelay and leaveDelay for custom timing (e.g., appear after 500ms, disappear after 200ms).
