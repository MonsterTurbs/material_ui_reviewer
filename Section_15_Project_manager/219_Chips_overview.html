💡 What Are Chips?

Chips are compact elements that represent inputs, attributes, or actions—think of little tags you can add or remove! 😊

📥 Importing Chips

Bring in the core component and any extras:

import Chip from '@material-ui/core/Chip';
import Avatar from '@material-ui/core/Avatar';
import FaceIcon from '@material-ui/icons/Face';

⚙️ Basic Usage

Render a simple chip with text:

<Chip label="Example Chip" />

🎨 Variants & Props

  • variant="outlined" for a border-only style.
  • size="small" or default size="medium".
  • color: default, primary, secondary.
  • disabled to gray out.

👤 Avatars & Icons

Add a circle avatar or icon:

<Chip
  avatar=<Avatar>A</Avatar>
  label="With Avatar"
/>
<Chip
  icon=<FaceIcon />
  label="With Icon"
/>

❌ Deletable Chips

Add onDelete to show an "X" and handle removal:

<Chip
  label="Removable"
  onDelete={() => console.log('Deleted!')}
/>

🔄 Clickable Chips

Use onClick for action when chip is clicked:

<Chip
  label="Click Me"
  onClick={() => alert('Chip clicked!')}
/>

📊 Dynamic Rendering

Render multiple chips from an array:

const data = ["Apple", "Banana", "Cherry"];
return (
  data.map(item => (
    <Chip key={item} label={item} onDelete={() => remove(item)} />
  ))
);

🔍 Component API

For all props (like clickable, avatar, custom delete icon), check the API tab under Chips in the Material-UI docs.
