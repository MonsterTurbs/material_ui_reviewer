1. Clear Form After Adding ➕✨

After you add a project, reset all your dialog fields so the form is blank next time:

// inside addProject() after setRows and close dialog
setName('');
setDate(new Date());
setTotal('');
setService('');
setComplexity('');
setUsers('');
setPlatforms([]);
setFeatures([]);

2. Disable “Add Project” Until Complete 🛑

Prevent duplicate or incomplete entries by disabling the button until all required answers are filled.

3. Website vs. Other Services 🌐📱

- If service === 'website', you only need:

  • Name
  • Total
  • At least one Feature
- For 'mobile app' or 'custom software', you also need:
  • Complexity
  • Users
  • Platforms

4. Disable Logic Example 💡

disabled={
  service === 'website'
    ? (
        name.length === 0 ||
        total.length === 0 ||
        features.length === 0
      )
    : (
        name.length === 0 ||
        total.length === 0 ||
        features.length === 0 ||
        service.length === 0 ||
        complexity.length === 0 ||
        users.length === 0 ||
        platforms.length === 0
      )
}

5. Test It Out ✔️

1. Open “Add Project” dialog.
2. Try leaving a required field empty — button stays disabled.
3. Fill all needed fields (depending on service) — button enables.
4. Click “Add Project” and see your entry in the table! 🎉
