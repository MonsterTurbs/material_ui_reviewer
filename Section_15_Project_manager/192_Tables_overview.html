📊 What Is a Material UI Table?

A Table shows rows and columns of data clearly. It’s super flexible and perfect for lists, reports, or any data you want to display! 🤓

🔧 Core Table Components

  • TableContainer: Box that holds your table.
  • Table: The main table element.
  • TableHead & TableRow: Header row with labels.
  • TableBody & TableRow: Your data rows.
  • TableCell: Individual cells in head or body.

📝 Simple Table Example

  • Define a rows array of objects (e.g., name, calories, fat).
  • In your JSX:
    <TableContainer component={Paper}> <Table> <TableHead> <TableRow> <TableCell>Name</TableCell> ... </TableRow> </TableHead> <TableBody> {rows.map(row => ( <TableRow key={row.name}> <TableCell>{row.name}</TableCell> ... </TableRow> ))} </TableBody> </Table> </TableContainer>

📏 Dense Table Option

For a tighter look, add size="small" to your <Table>:

<Table size="small"> ... </Table>

🔀 Sorting & Filtering

Use the advanced example to:

  • Click column headers to sort A→Z or Z→A. 🔠
  • Filter rows by category (e.g., only show "Websites"). 🔍
  • Map over data and update state to drive filters/sorting.

✅ Selection & Deletion

  • Add checkboxes in each row to select items. ☑️
  • Show a toolbar or Snackbar with "Undo" to reverse deletions. ↩️

📑 Pagination & Sticky Header

  • Use TablePagination for page controls. ⏭️
  • Set table container style for stickyHeader to keep headers in view. 📌

🧩 Spanning & Virtualization

  • Use colSpan or rowSpan on TableCell for merged cells. ↔️
  • For huge data sets (>200 rows), consider react-virtualized to render only visible rows. 🚀
