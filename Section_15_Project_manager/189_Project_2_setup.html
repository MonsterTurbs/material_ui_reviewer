🛠️ Step 1: Get Next.js Boilerplate

  • 📄 Go to Material-UI → Guides → Server-side Rendering → Next.js example.
  • ⚙️ Copy the curl command.
  • 📂 In your terminal, cd to your desired folder, then paste the curl.
  • 📦 cd nextjs & npm install to fetch dependencies.

➕ Step 2: Install MUI Packages

npm install @material-ui/styles @material-ui/icons

📂 Step 3: Add Shared Files

  • ✔️ Download “project two” files (header.js, footer.js, theme.js).
  • 📋 Copy into src/UI/ folder.
  • 🗑️ Remove sample content (e.g., protip logos, unused imports).

🔄 Step 4: Update Imports

  • In _app.js & _document.js, import theme from src/UI/theme.
  • In header/footer, import Link from next/link (not react-router).
  • Clean up unused imports and logic in header.

🖼️ Step 5: Copy Assets

  • Create public/assets directory.
  • Copy footer adornment SVG into public/assets/.
  • Update image src in footer to /assets/footer-adornment.svg.

📝 Step 6: Scaffold Index Page

  • In pages/index.js, clear default code.
  • Import React, Grid, Typography, makeStyles, useTheme.
  • Build basic layout: <Grid container direction="column"><Typography>Projects</Typography></Grid>.

🔗 Step 7: Add Google Fonts

  • Copy Google Fonts link (Railway & Roboto) into _document.js in the head.

🚀 Step 8: Run Dev Server

npm run dev
Open http://localhost:3000 to see your blank “Projects” page with header & footer.

✅ You’re Ready!

With boilerplate set, next up: building search, filters, table, dialog, and more. Let’s dive in!
