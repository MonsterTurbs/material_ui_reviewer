⚪ Match Background & Remove Shadow

  • Apply pure white to page body in _document.js: <body style="background-color: #fff"> 😊
  • Remove table shadow by setting elevation={0} on TableContainer.

🟦 Style Header Cells

  • In theme.js overrides, target MuiTableCell.head:
    • MuiTableCell: {
  head: {
    fontSize: '1rem',
    fontWeight: 700,
    color: '#1976d2',
    borderColor: '#1976d2',
    borderWidth: 2,
  },
},

⚪ Style Body Cells

  • Also in overrides, target MuiTableCell.body:
    • MuiTableCell: {
  body: {
    color: '#666',
    borderColor: '#1976d2',
    borderWidth: 2,
  },
},

🎯 Center & Wrap Columns

  • Add align="center" to every <TableCell> for centering text.
  • Limit long cells (e.g. Features) with: style={{ maxWidth: '5rem', whiteSpace: 'normal' }} to wrap text. ↔️

🔻 Footer Spacing

  • Push footer down by adding style={{ marginBottom: '15rem' }} to the Grid item wrapping your table.

✅ Test & Tweak

  • Run npm run dev and refresh localhost:3000.
  • Verify colors, alignments, and wrap behavior match your design! 🚀
