📋 Goal

We want to let users delete selected rows from our table and give them the ability to undo that action.

🗑️ Setting Up Delete Icon

Wrap the delete icon button with an onClick handler: <IconButton onClick={handleDelete}>.

✅ Tracking Selections

We store selected rows in state: const [selected, setSelected] = useState([]); and pass selected and setSelected as props to the toolbar.

🖥️ Logging Selected Rows

Inside handleDelete, do console.log(props.selected) to see which rows are selected (e.g., names or objects).

🔍 Filtering Selected Rows

Use rows.filter(row => props.selected.includes(row.name)) to get only the rows user wants to delete.

🧩 Hiding vs Deleting

Instead of removing data immediately, we set row.search = false to hide it, allowing undo before permanent delete.

🔄 Shallow Copy Trick

Create a shallow copy of rows: const newRows = [...props.rows];, then modify objects inside selectedRows.map(row => row.search = false).

⚙️ Updating State

After hiding, call props.setRows(newRows) to update table display.

🔗 Passing Props

Ensure rows and setRows are passed from parent to EnhancedTableToolbar and used correctly.

⏪ Undo Feature

Next, we'll add a snackbar or button to reverse the search=false changes if the user clicks 'Undo'.
