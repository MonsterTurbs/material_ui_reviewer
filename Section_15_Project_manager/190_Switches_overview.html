🔀 What Is a Switch?

A Switch is a toggle control that you can turn on or off. It’s just like a light switch you see on your phone or in settings! 💡

📋 When to Use Switches

  • 🔲 Checkboxes: for choosing multiple options.
  • 🔘 Radio buttons: for choosing exactly one option.
  • ⚙️ Switches: for toggling a setting on/off (like filters).

🛠️ How to Add a Switch

First, import the components:

import Switch from '@material-ui/core/Switch';
import FormGroup from '@material-ui/core/FormGroup';
import FormControlLabel from '@material-ui/core/FormControlLabel';

🎨 Customization & Accessibility

  • 📏 Sizes: set size="small" or default.
  • 🎨 Colors: use props like color="primary".
  • 🔈 A11y: add inputProps={{ 'aria-label': '...'}} for screen readers.
