Select Component 🍱

A Select lets users pick one or more options from a list. It’s perfect when you have many choices and want a dropdown.

Basic Single Select ✅

Use the simple Select for exactly one choice. It shows the selected value in the input, and clicking it opens a menu of <MenuItem>s.

FormControl + InputLabel 🏷️

Wrap your <Select> in a <FormControl> and add an <InputLabel>. Use the same id on both so the label floats correctly.

MenuItem List 📋

Inside your <Select>, place one <MenuItem value="..."/> for each option. The value gets passed to your state onChange.

Multiple Select 📝

Add the multiple prop to allow picking more than one option. The value becomes an array of selected items. Great for “pick all that apply” questions.

Native Select 🌐

Use the native prop for the browser’s built-in dropdown. It can feel more natural on mobile or simple forms, but has less styling control.

Styling & Variants 🎨

You can switch between outlined, filled, or standard variants. Add disabled or error props to show different states.

Programmatic Control ⚙️

You can control open/close by using the open prop and manage it in state, plus onClose and onOpen callbacks.

Grouped Options 🗂️

If you have many items, use <ListSubheader> inside your menu to group related choices under headings.

Accessibility 🔒

Always give your <Select id="my-select"> and its <InputLabel htmlFor="my-select"> matching IDs so screen readers announce them properly.
