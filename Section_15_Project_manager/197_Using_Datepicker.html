📦 Install Pickers & Date Library

  • Stop dev server and run in your project folder:
  • npm install @material-ui/pickers @date-io/date-fns date-fns@next ✅

📥 Import Components

At the top of Index.js, add:

import Dialog from '@material-ui/core/Dialog';
import DialogContent from '@material-ui/core/DialogContent';
import { MuiPickersUtilsProvider, KeyboardDatePicker } from '@material-ui/pickers';
import DateFnsUtils from '@date-io/date-fns';

🔐 Dialog State

Control the dialog open/close:

const [dialogOpen, setDialogOpen] = useState(false);

🚪 Add the Dialog

  • Wrap content with:
    • <Dialog open={dialogOpen} onClose={() => setDialogOpen(false)}>
  ...dialog content...
</Dialog>
  • Trigger open with your Add button:
    <AddIcon style={{cursor: 'pointer'}} onClick={() => setDialogOpen(true)} /> 🎉

🛡️ Wrap with Pickers Provider

At a high level (e.g. above your main grid), wrap with:

<MuiPickersUtilsProvider utils={DateFnsUtils}>
  ...your app...
</MuiPickersUtilsProvider>

🗂️ Dialog Title

Inside Dialog, show a centered title:

<Grid container justify="center">
  <Grid item>
    <Typography variant="h1" gutterBottom>Add a New Project</Typography>
  </Grid>
</Grid>

✏️ Project Name Field

First question in dialog:

const [name, setName] = useState('');

<TextField
  label="Name"
  id="name"
  value={name}
  onChange={e => setName(e.target.value)}
/>

📅 Date Picker Field

Second column: pick a date!

const [date, setDate] = useState(new Date());

<KeyboardDatePicker
  format="MM/dd/yyyy"
  value={date}
  onChange={d => setDate(d)}
/>
