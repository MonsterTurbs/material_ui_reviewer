üî≤Page Layout

Use a Grid container with direction="row":

  • Left Grid item: column direction for title and Lottie.
  • Right Grid item: column direction for the questions and buttons.

üß†State Management

Store questions in state as an array of objects. Each object is a question with its own details.

This lets React re-render when values change (like switching questions or selecting options).

‚ùìQuestion Object

Each question has:

  • id: unique number (e.g., 1, 2, ...).
  • title: the question text (e.g., "Which service are you interested in?").
  • active: true if this question is showing.
  • Optional subtitle: extra hint (e.g., "Select one").
  • options: an array of option objects.

üîòOption Object

Each option inside a question has:

  • id: unique number.
  • title and optional subtitle.
  • icon path and iconAlt.
  • selected: true if the user picked it.
  • cost: number for price calculation (some start at 0).
