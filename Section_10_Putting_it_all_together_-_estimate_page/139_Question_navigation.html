➕Adding More Questions

Copy softwareQuestions and websiteQuestions constants from the provided text file into Estimate.js. These extend your initial defaultQuestions.

🗂️Question Sets Explained

  • 🔹defaultQuestions: first screen choices (services).
  • 🔹softwareQuestions: questions for Custom Software & App Development.
  • 🔹websiteQuestions: questions for Website Development only.

This separation ensures at the end you only process selected options for the chosen service. 🎯

↔️Navigation Logic

Make arrows clickable to move between questions:

  1. On click, clone the current questions array to avoid mutating state.
  2. Set active to false on current question, and true on next/previous.
  3. Update state with the new array to re-render the component. 🔄

⚠️Why Clone State?

React state must be immutable. Nested arrays/objects require deep copying to avoid mutating originals.

Simple shallow copies still reference inner objects, leading to unexpected changes.

📦Using Lodash cloneDeep

  1. Install: npm install --save lodash
  2. Import: import { cloneDeep } from 'lodash';
  3. Clone: const newQuestions = cloneDeep(currentQuestions);

This ensures full deep copy of nested question objects. 🛡️
