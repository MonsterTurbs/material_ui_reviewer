✅ Study Notes: Course Requirements

🖥️ Computer Setup

You will need a computer to take this course. You can use:

  • 💻 Mac
  • 🖥️ Windows

The final project will look the same on both platforms. The instructor is using a Mac but will note differences for Windows users.

📝 Code Editor (Your Text Editor)

You can use any code editor you like! Here are the top suggestions:

  • Atom (Free, used in the course, lots of helpful plugins)
  • VS Code (Free, from Microsoft, great for Windows users)
  • Sublime Text (Popular on Mac, may require payment)

Don't worry if you’re using something else — as long as it can edit code, you’re good!

📚 React Knowledge Needed

This course is not for total beginners in React. You should already know the basics:

  • What are components
  • How JSX works
  • How to use a Node.js environment

The instructor will explain things as the course goes on, but not at the beginner level for React.

🚀 Ready to Begin?

If you have a computer, a code editor, and basic React knowledge — you're ready! 🎉
