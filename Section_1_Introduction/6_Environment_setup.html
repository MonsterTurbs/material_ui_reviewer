🚀 Study Notes: Setting Up Your React + Material UI Project

📦 Step 1: Check Your Node.js Version

In your terminal, run:

node -v

If you're not using version 10.16.3 or later, go to nodejs.org and download the latest version.

🛠️ Step 2: Create a New React App

Use NPX to start a new React project:

npx create-react-app arc-development

This sets up all the files needed to build a React app.

📁 Step 3: Navigate to the Project Folder

cd arc-development

Now you're inside the project folder and ready to add more tools!

🧭 Step 4: Install React Router

This lets us switch between pages in our app:

npm install --save react-router-dom

Ignore warnings about TypeScript — it's not required for this course.

🎨 Step 5: Install Material UI

Install the core Material UI library:

npm install --save @material-ui/core

Also, add the Roboto font to your HTML:

<link href="https://fonts.googleapis.com/css?family=Roboto&display=swap" rel="stylesheet">

💻 Step 6: Open the Project in Your Code Editor

Use any editor like Atom, VS Code, or Sublime. Open the index.html file in the public folder and add the Roboto font link.

🔘 Step 7: Add a Test Material UI Button

In App.js, import and use a button to test if everything works:

import Button from '@material-ui/core/Button';

Add inside the return:

<Button variant="contained">Click me</Button>

▶️ Step 8: Run the Project

Go back to your terminal and start the server:

npm start

Your browser should open and show the "Click me" button with the Material UI ripple effect.

🎉 You're Ready!

Congrats! Your setup is done and your first Material UI project is running. Let's keep going to build more amazing things! 👨‍💻👩‍💻
